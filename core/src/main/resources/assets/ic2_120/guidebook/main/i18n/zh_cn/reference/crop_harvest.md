# 作物收获产出参考

## 收获机制

每次收获时，掉落判定次数服从以 `dropChance` 为均值的高斯分布：

```
dropChance = dropGainChance × 1.03^Ga
掉落次数 = round( 正态分布(均值=dropChance, 标准差=dropChance) )
```

每次判定调用 `createGainStacks()` 得到一个基础产出堆，然后有 `Ga%` 概率使该堆数量 +1。
