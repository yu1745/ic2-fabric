# 流体运输系统

IC2 中有两套流体传输机制，用途不同，互不依赖：

- **管道与泵附件**：长距离运输，通过铺设管道网络把流体从 A 点送到 B 点
- **机器流体升级**：相邻交互，机器自动与旁边的容器交换流体，不走管道

---

## 管道与泵附件（长距离运输）

### 管道

管道铺设流体传输路径。有 4 种尺寸 × 2 种材质 = 8 种管道。

#### 尺寸与材质

| 尺寸 | 基础流量 | 半径 |
|------|---------|------|
| 微型（tiny） | 0.4 桶/秒 | 2/16 格 |
| 小型（small） | 0.8 桶/秒 | 3/16 格 |
| 中型（medium） | 2.4 桶/秒 | 4/16 格 |
| 大型（large） | 4.8 桶/秒 | 5/16 格 |

| 材质 | 倍率 | 颜色 |
|------|------|------|
| 青铜 | ×1 | 铜褐色 |
| 碳纤维 | ×2 | 深黑色 |

**实际流量 = 基础流量 × 材质倍率**

#### 所有管道

| 物品 | 流量 | 配方（核心材料） |
|------|------|-----------------|
| 微型青铜管道 | 0.4 桶/秒（400 mB/s） | 青铜外壳 ×3 → 6 个 |
| 小型青铜管道 | 0.8 桶/秒（800 mB/s） | 青铜外壳 → 3 个 |
| 中型青铜管道 | 2.4 桶/秒（2400 mB/s） | 青铜板 ×3 → 2 个 |
| 大型青铜管道 | 4.8 桶/秒（4800 mB/s） | 青铜板 → 1 个 |
| 微型碳纤维管道 | 0.8 桶/秒（800 mB/s） | 碳纤维 ×3 → 6 个 |
| 小型碳纤维管道 | 1.6 桶/秒（1600 mB/s） | 碳纤维 → 3 个 |
| 中型碳纤维管道 | 4.8 桶/秒（4800 mB/s） | 碳网 ×3 → 2 个 |
| 大型碳纤维管道 | 9.6 桶/秒（9600 mB/s） | 碳网 → 1 个 |

#### 连接

- 管道自动连接到相邻的管道、储罐、机器等（只要有流体 API 的方块）
- **扳手右键点击某面**：切换该面的连接开关。关闭后该面不再连接任何东西
- **扳手 Shift+右键**：切换透明模式，可以看到内部流体流动

---

### 泵附件

泵附件是管道的**抽取端**。管道网络**必须**通过泵附件才能从储罐或机器中抽取流体。没有泵附件，管道只是通路，不会从任何地方抽东西。

| 物品 | 流量 | 说明 |
|------|------|------|
| 青铜泵附件 | 0.4 桶/秒（400 mB/s） | 基础型，流量等效于本材质的微型管道 |
| 碳纤维泵附件 | 0.8 桶/秒（800 mB/s） | 高速型，流量等效于本材质的微型管道 |

#### 放置

泵附件有方向性，必须**正面朝向目标方块**放置：

```
 储罐/机器    ←   泵附件    ←   管道
（有流体）   正面      背面
```

- **正面**（朝向目标的面）：只连接非管道的流体容器（储罐、机器），**不**连接其他管道
- **背面和侧面**：连接其他管道，构成传输路径

#### 过滤

右键点击泵附件打开 GUI，可以设置**过滤样本**：

- 手持含流体的容器（桶、流体单元等）点击过滤槽 → 设置过滤，泵附件只抽取该种流体
- 空手点击过滤槽 → 清除过滤，泵附件抽取任意流体

#### 常见用途

- **从焦炭窑炉篦下抽杂酚油**：泵附件贴在炉篦正下方，正面朝上→炉篦，背面接管→储罐或半流质发电机
- **从储罐抽水**：泵附件贴在储罐侧面，接管到需要水的地方
- **从机器抽产物**：泵附件贴在机器的流体输出面，接管到储罐

### 管道传输机制

#### 工作流程

1. **找提供者**：网络扫描所有泵附件，每个泵附件正面朝向的容器就是一个流体来源
2. **找接收者**：网络扫描所有非泵附件的连接面，找到能接收流体的容器
3. **路径查找**：为每个接收者找到最近的提供者路径
4. **传输**：从提供者抽取，沿管道路径，注入接收者

全部基于 Fabric 流体 API（`FluidStorage.SIDED`），因此支持任何实现了该 API 的第三方容器。

运输系统不会把流体全局划分为“气体”和“液体”。水、蒸汽、压缩空气以及第三方流体都统一以 Fabric `FluidVariant` 传输；每台机器或储罐通过自己的存储规则决定接受哪些具体流体。

#### 混流保护

同一网络在同一 tick 内只传输**一种流体**。如果多个泵附件抽取不同种类的流体：

- 整网停机
- 需要清除多余的流体来源才能恢复

## 机器流体升级（相邻交互）

与管道网络无关，机器安装**流体弹出升级**或**流体抽入升级**后，每次 tick 自动与相邻容器交换流体。

| 升级 | 功能 |
|------|------|
| 流体弹出升级 | 机器自动把内部流体喷出到相邻容器 |
| 流体抽入升级 | 机器自动从相邻容器抽取流体 |

### 速率

每张升级提升速率（上限 4 张）：

| 升级数量 | 每 tick | 每秒 |
|---------|--------|------|
| 1 张 | 50 mB/t | 1000 mB/s |
| 2 张 | 200 mB/t | 4000 mB/s |
| 3 张 | 800 mB/t | 16000 mB/s |
| 4 张 | 3200 mB/t | 64000 mB/s |

### 配置

- **设置过滤**：手持升级 + 副手持含流体容器 + 右键
- **清除过滤**：副手为空 + 右键
- **设置方向**：潜行 + 右键循环切换方向

方向循环：下 → 上 → 北 → 南 → 西 → 东 → 全部方向

## Jade HUD

安装 Jade 后，悬停管道可查看：

- **流量**：当前流量 / 最大流量（mB/t）进度条
- **混流停机**：混流时显示停机状态
- **泵附件**：标签显示"泵附件"及过滤的流体类型
