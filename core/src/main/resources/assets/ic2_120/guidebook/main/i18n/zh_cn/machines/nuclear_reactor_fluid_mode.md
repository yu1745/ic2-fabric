# 流体冷却反应堆（流体模式）

<BlockImage id="ic2_120:reactor_vessel" scale="4" />

流体冷却反应堆是核反应堆的**流体模式**。它不直接发电，而是将热量转化为**热冷却液（HU）**，再由[流体热交换机](liquid_heat_exchanger.md)转回 EU 或其它能量。产热本身与 EU 模式一致——仅当使用 MOX 燃料棒且堆温超过最大热量的 50% 时，MOX 棒的产热才会翻倍。

本页介绍定义和运行流体模式外壳的四种方块。如需干式 **EU 模式**，见[核反应堆](nuclear_reactor.md)。

## 流体模式识别

当核反应堆被以反应堆为中心的 **5×5×5 核反应堆压力容器外壳**完整包围，且至少存在 **2 个反应堆流体接口**时，流体模式会**自动**识别。无需红石或 GUI 切换——外壳本身就切换模式。

### ⚠ 按产热配置流体热交换机数量

单个流体热交换机最多输出 **100 HU/t**（10 个热传导器 × 每个 10 HU/t），转换比为 **1,000 mB 热冷却液 → 20,000 HU**。这是单台机器的上限，但可以**并联多台流体热交换机**——每多一台就增加 100 HU/t 的处理能力。两台可处理最高 **200 HU/t**，三台最高 **300 HU/t**，以此类推。

危险在于反应堆的产热速度**超过了你所有流体热交换机的总处理能力**。此时流体热交换机把热冷却液转回凉冷却液的速度，**慢于反应堆的发热速度**，于是：

1. **热冷却液积压**在输出储罐和管道中——反应堆的流体接口无法再排出更多热冷却液。
2. **凉冷却液产生过慢**——回流管线无法提供足够的凉冷却液来跟上反应堆的发热速度。
3. 反应堆**无法散热**（凉冷却液供不应求），温度持续上升。
4. 超过 **10,000 HU** 后反应堆**爆炸**。

因此请按燃料棒负载配置流体热交换机的数量：算出所需台数，使其总处理能力（每台 ×100 HU/t）覆盖反应堆的产热速度。

## EU 模式 vs 流体模式

| 属性 | EU 模式 | 流体模式 |
|------|------|------|
| 产热 | 正常 | 正常（仅 MOX 堆温 > 50% 时翻倍） |
| 能量输出 | 正常 | **无电力**（改为输出 HU） |
| 散热片自散热 | 降低反应堆温度 | **加热冷却液** |
| 流体需求 | 无 | 冷却液 + 热冷却液 |
| 散热片修复 | 被动 | **需要冷却液** |

## 外壳方块

### 核反应堆压力容器

构建 5×5×5 外壳的壳体材料。它纯属结构用途——本身不导电也不导流。外壳必须**完整无缺**，反应堆才会进入流体模式；任何缺口都会禁用流体模式，使反应堆回退到 EU 模式行为。

### 反应堆流体接口

循环冷却液。每个接口内置 **16,000 mB** 储罐。至少需要 **2 个接口**：一个接入**冷却液**，一个排出**热冷却液**。用管道将它们接到[流体热交换机](liquid_heat_exchanger.md)，即可把热量回收为 HU。

- 转换比：**20,000 HU → 1,000 mB 热冷却液**
- 只有散热片的**自散热**部分计入冷却液转换。

### 反应堆访问接口

流体模式下，核反应堆核心被压力容器外壳完整包裹，因此**无法直接右键反应堆**打开其界面。访问接口是打开反应堆界面的唯一途径——改为右键访问接口即可。它同时提供自动化的物品输入/输出，用于插入和提取燃料棒、枯竭燃料棒及其它内部组件。可安装在外壳的任意面上。

### 反应堆红石接口

在 EU 模式下，反应堆结构的**任意方块**都能接收红石信号；但在流体模式下，压力容器外壳会阻挡这一特性——**只有红石接口才能接收红石信号**。这是两种模式的本质区别。在外壳上放置一个红石接口，再连上拉杆、按钮或红石电路，即可实现**紧急停机**或基于脉冲的间歇运行等自动化控制。

## 热量守恒

```
燃料棒产热（MOX 堆温 > 50% 时该棒产热 ×2）
→ 热量分配
→ 散热片自散热（散失热量）
→ 加热冷却液
→ 未转换的热量回到散热片/反应堆
```

## 常数

| 常数 | 数值 |
|------|------|
| 冷却液储罐容量 | 16,000 mB |
| 每桶冷却液对应的 HU | 20,000 HU |
| 外壳要求 | 完整的 5×5×5 压力容器 |
| 进入流体模式的最少流体接口 | 2 |
