# 采矿机

<BlockImage id="ic2_120:miner" p:facing="north" scale="4" />

采矿机是一种自动化挖掘机器，可在其下方指定范围内挖掘矿石和方块。它需要采矿管道来向下延伸，以及 EU 来运行。

## 工作方式

采矿机向下钻孔，消耗其库存中的**采矿管道**以延长钻探深度。它会在采矿半径内开采任何可破坏的方块，并将所有掉落物收集到输出栏位中。当管道触及底部后，采矿机会开始在配置的半径内水平挖掘。

## 扫描仪

采矿机支持 OD（方向探测器）和 OV（范围扩展）扫描仪来控制采矿区域：

- **OD 扫描仪**：将采矿范围限制在采矿机正下方的指定半径内。
- **OV 扫描仪**：扩展扫描范围，使采矿机覆盖更大的区域。

## 需求

- **EU**：为采矿机构提供电力。
- **采矿管道**：随着采矿机向下钻探而消耗。
- **扫描仪**（可选）：用于控制采矿区域。

## 使用

将采矿机放置在待采矿区域上方。提供采矿管道和 EU 电力。采矿机会自动在其范围内挖掘矿石和石头，无需手动挖掘即可获得稳定的资源供应。
