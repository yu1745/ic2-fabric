# 船

<ItemImage id="ic2_120:rubber_boat" scale="4" />

船系列是一套由 IC2 材料制成的四艘水上载具。其中三艘——橡皮艇、破损的橡皮艇和碳纤维船——是原版风格的划艇，区别在于船体材质与操控手感；电动艇则是由 mEU 驱动的载具，用于高速、电机辅助的航行。四艘船都能放在水面上，使用方式与原版船完全相同。

## 属性

| 类型 | 材料 | 速度等级 | 配方 | 说明 |
|------|----------|:----------:|--------|-------|
| 橡皮艇 | 橡胶 | 原版风格 | `"   " / "R R" / "RRR"`（R = 橡胶） | 基础水上载具，划艇的基线版本 |
| 破损的橡皮艇 | 橡胶（已损坏） | 原版风格 | 无序配方——只能被修复回橡皮艇 | 用于修复已损坏的橡皮艇 |
| 碳纤维船 | 碳板 | 原版风格 | `"   " / "P P" / "PPP"`（P = 碳板） | 碳纤维轻质船体 |
| 电动艇 | 铁板 + 绝缘铜缆 + 电动马达 + 铁转子 | 电动（mEU 驱动） | `"CCC" / "IMI" / "IRI"`（C = 绝缘铜缆，I = 铁板，M = 电动马达，R = 铁转子） | 比其他三艘更快，但造价更高 |

前三艘是纯船体，没有搭载任何动力系统。电动艇则带有真正的电动机和转子，描述上是电机驱动且比其他三艘更快。具体的速度与操控数值定义在生成的实体上（见相关章节）；如果需要精确数值，请参考 `ModEntities` 中实体的行为定义，而不是物品本身。

## 使用方式

### 放置

- 手持船物品**对着水面右键**即可生成可骑乘的船实体。
- 放置使用的是原版船相同的标准射线检测：只有命中方块、玩家视线畅通、且有足够空间容纳实体的碰撞箱时才会成功放置。
- 实体类型与物品一一对应：橡皮艇 → `ModEntities.RUBBER_BOAT`，破损的橡皮艇 → `ModEntities.BROKEN_RUBBER_BOAT`，碳纤维船 → `ModEntities.CARBON_BOAT`，电动艇 → `ModEntities.ELECTRIC_BOAT`。
- 在**生存模式**下，物品会在放置成功时被**消耗**（每艘船 1 个）。在**创造模式**下，物品不会被消耗。
- 物品只作为生成介质——实体一旦放置，物品就消失了，船的行为完全由实体决定。

### 上船与乘坐

- 对着已放置的船右键即可上船，与原版船完全一致。
- 每艘船实体都有各自的速度和操控特性，定义在实体侧；如果需要确切数值，请查阅实体定义。
- 使用标准下船键离开船。

### 修复破损的橡皮艇

橡皮艇受到损坏后会变成**破损的橡皮艇**。要将其修复回可用的橡皮艇：

- 在工作台中（无序合成）将破损的橡皮艇与 **1 个橡胶** 组合。
- 输出为全新的橡皮艇，并消耗掉破损的那一个。

这是唯一使用到破损的橡皮艇物品的配方。

### 给电动艇充电

电动艇是一台电力设备——它内置真正的电动机和转子，设计上由 mEU 驱动。描述上它是电机驱动且比三艘非电动船更快，但具体的续航/速度/耗电数值定义在实体侧（见相关）。把它当作一台电动载具来对待：搭建一个充电装置（充电器、充电板或任何等级合适的 EU 储能设备）来维持它的运行。

## 配方

四艘船中有三艘有有序合成配方。破损的橡皮艇**不能**直接合成——它作为橡皮艇的损坏状态出现，并通过无序合成修复配方来还原。

|:-----------:|:-----------:|
| <Recipe id="ic2_120:rubber_boat" /> | <Recipe id="ic2_120:carbon_boat" /> |
| 电动艇 | 破损的橡皮艇（修复） |
| <Recipe id="ic2_120:electric_boat" /> | <Recipe id="ic2_120:repair_to_rubber_boat" /> |

配方说明：

- **橡皮艇** 使用 `Compat.Items.RUBBER` 标签，因此接受任意模组的橡胶物品。
- **碳纤维船** 直接使用碳板物品。
- **电动艇** 使用绝缘铜缆物品、`Compat.Items.PLATES_IRON` 标签（铁板）、电动马达物品和铁转子物品。
- **破损的橡皮艇 → 橡皮艇** 是一个无序配方：1 个破损的橡皮艇 + 1 个橡胶 → 1 个橡皮艇。
