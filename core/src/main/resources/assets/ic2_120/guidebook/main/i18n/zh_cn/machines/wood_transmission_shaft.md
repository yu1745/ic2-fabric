# 木制传动轴

<BlockImage id="ic2_120:wood_transmission_shaft" scale="4" />

木制传动轴用于沿直线传输动能（KU）。它是最简单、最低等级的传动轴，适用于低功率动能系统。

## 吞吐量

木制传动轴最多可承受 **32 KU** 的动能功率。

## 使用方式

- 将传动轴排成一条直线，用于从动能发电机向动能机器传输动能。
- 相邻放置时，动能会从一个传动轴传递到下一个。
- 木制传动轴在六个面均可连接。
- 更高级的传动轴（铁制、钢制、碳纤维）可承受更高的 KU，用于更高要求的应用。
