# 钢制摇杆

<ItemImage id="ic2_120:steel_crank_handle" scale="4" />

钢制摇杆是用于手摇动能发生机的高级摇杆。安装后可使机器在摇动时产生 256 KU/t。

钢制摇杆提供了可观的动能输出，适合需要大量动能的中后期工业生产线。
