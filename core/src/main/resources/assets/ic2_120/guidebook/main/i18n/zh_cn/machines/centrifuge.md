# 热能离心机

<BlockImage id="ic2_120:centrifuge" p:facing="north" p:active="true" scale="4" />

热能离心机通过高速旋转处理纯净粉碎矿石或其他材料，将其分离为多种精炼产物。它是 IC2 矿石加工链中继洗矿机之后的第三步。

## 输出

- **EU 存储：** 10,000 EU
- **等级：** 2
- **能量消耗：** 每次操作约 50 EU/t

## 槽位

- 左侧槽位：输入（纯净粉碎矿石）
- 右侧槽位：多种输出（取决于材料）
