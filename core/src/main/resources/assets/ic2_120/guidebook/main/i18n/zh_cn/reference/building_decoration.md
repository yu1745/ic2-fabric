# 建筑与装饰

## 橡胶木

橡胶树提供完整的木质方块系列：橡胶原木、橡胶木、去皮橡胶原木、去皮橡胶木、橡胶木板、楼梯、台阶、门、活板门、按钮、压力板、栅栏、栅栏门、告示牌。

## 强化建筑

- 防爆石、防爆玻璃、防爆门适合保护高价值机器和核电区域。
- 铁栅栏可用于工业风格围栏与隔离。
- 反应堆压力容器属于核电结构方块，见：[核电方块与组件](nuclear_components.md)。

## 建筑泡沫

- 建筑泡沫可通过建筑泡沫桶、单元、喷枪或机器链路使用。
- 强化建筑泡沫用于更坚固的结构。
- 泡沫墙有 16 种染色版本：白、橙、品红、浅蓝、黄、黄绿、粉、灰、浅灰、青、紫、蓝、棕、绿、红、黑。
- 树脂垫、橡胶垫、羊毛垫是薄片类装饰/功能方块。

## 脚手架

- 木质脚手架和铁质脚手架用于临时施工。
- 强化木质脚手架和强化铁质脚手架更适合长期结构。
- 采矿管道属于[采矿机](../machines/miner.md)的工作链路，不是普通流体管道。

## 杂项物品

- <ItemLink id="ic2_120:itnt" />（工业 TNT）——比原版 TNT 威力大得多的爆炸物。爆炸半径和伤害显著更高，适合大规模拆除和采矿。使用时请格外小心。
- <ItemLink id="ic2_120:coin" />（工业硬币）——通过 <ItemLink id="ic2_120:metal_former" />（挤出模式）将铁外壳加工而成的工业货币。硬币在某些 IC2 配方中使用。
