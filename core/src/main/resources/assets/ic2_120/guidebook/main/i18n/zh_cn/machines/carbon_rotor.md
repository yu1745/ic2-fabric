# 碳纤维转子

<ItemImage id="ic2_120:carbon_rotor" scale="4" />

碳纤维转子是用于风力动能发生机和水力动能发生机的最高级转子。

- **风力**：半径 5，倍率 4×
- **水力**：半径 2.5，倍率 4×

这是最佳性能的转子，拥有最大的旋转半径和最高的输出倍率，能够最大限度地从风力和水力中获取动能。
