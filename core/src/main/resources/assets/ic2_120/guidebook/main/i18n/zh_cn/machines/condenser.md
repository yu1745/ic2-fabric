# 冷凝器

<BlockImage id="ic2_120:condenser" p:facing="north" p:active="true" scale="4" />

冷凝器通过冷却将蒸汽转化为蒸馏水。它接受 EU 输入，当内部缓冲区满时会排出多余能量。它是 IC2 热力加工链的一部分。

## 输出

- **EU 存储：** 100,000 EU
- **等级：** 3
- **转化：** 蒸汽 → 蒸馏水
- **溢出：** 储能满时排放多余 EU
