# 橡胶树与世界资源

## 矿石

世界生成提供锡、铅、铀、泥炭等资源，部分矿石有深层变种。铱矿石较稀有，是后期材料链的关键资源。

## 橡胶树

- 橡胶树会生成橡胶原木、橡胶树叶和橡胶树苗。
- 橡胶原木侧面可能带树脂孔。
- 用木龙头或电动树脂提取器右键树脂孔可获得粘性树脂。
- 粘性树脂可加工为橡胶，用于导线绝缘、工具、护甲和机器部件。

## 橡胶木方块

橡胶木提供完整木质方块：原木、去皮原木、木板、楼梯、台阶、门、活板门、按钮、压力板、栅栏、栅栏门、告示牌等。

## 早期资源路线

1. 挖锡、铜、铅、铀矿，准备基础金属链。
2. 找橡胶树，收集树脂和树苗。
3. 制作锻造锤、板材切割剪刀、基础导线。
4. 建立发电机、储电箱和第一批电动机器。
