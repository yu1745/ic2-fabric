# 模式扫描机

<BlockImage id="ic2_120:uu_scanner" p:facing="north" scale="4" />

模式扫描机（UU 扫描机）分析物品并创建数字模式，这些模式可以存储起来供后续复制使用。这是 UU 物质复制过程的第一步。

## 能量与存储

- **能量等级：** 3
- **最大 EU 输入：** 512 EU/t
- **EU 存储：** 200,000 EU
- **扫描时间：** 3,300 tick / 165 秒
- **耗电：** 256 EU/t
- **总扫描耗电：** 844,800 EU

## 工作方式

扫描机只会扫描配置中存在复制模板的物品。它还需要一个明确的模式保存目标：要么旁边有唯一一个模式存储机，要么在没有相邻模式存储机时，水晶槽中放有模式水晶。

扫描完成后会消耗 1 个输入物品，并把模板暂存在扫描机内部。使用扫描机按钮把缓存模板保存到模式水晶或相邻模式存储机，也可以删除它。缓存模板尚未处理时，扫描机不会开始新的扫描。

## 槽位

- **输入槽：** 要扫描的物品
- **电池槽：** 3 级电池放电
- **水晶槽：** 模式水晶目标

扫描机没有升级槽。物品自动化可以插入扫描输入、电池和模式水晶；三个槽位都可抽出。

## 使用

1. 在旁边放置模式存储机，或插入模式水晶。
2. 放入白名单物品并提供持续 EU。
3. 扫描完成后，把模板保存到存储机或水晶。
4. 在模式存储机中选择模板，供复制机使用。

每种物品只需扫描一次。复制时仍然会消耗该模板记录的 UU 物质成本。
