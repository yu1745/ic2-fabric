# 模式扫描机

<BlockImage id="ic2_120:uu_scanner" p:facing="north" scale="4" />

模式扫描机（UU 扫描机）分析物品并创建数字模式，这些模式可以存储起来供后续复制使用。这是 UU 物质复制过程的第一步。

## 能量与存储

- **EU 存储**：200,000 EU
- **等级**：3 级（HV，1024 EU/packet）

## 工作方式

将要复制的物品放入扫描机的输入槽中，即可开始分析其分子结构。扫描完成后，会生成该物品的模式。该模式可以传输到**模式存储机**，供**复制机**使用。

## 使用

1. 将想要复制的物品放入扫描机中。
2. 提供足够的 EU 以完成扫描过程。
3. 扫描完成后，将模式存储机放在扫描机旁边以接收模式数据。
4. 该模式即可供复制机使用。

每种物品只需扫描一次。模式存储后，您可以无限次复制该物品（只要拥有足够的 UU 物质和 EU）。
