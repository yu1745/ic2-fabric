# Brush

<ItemImage id="ic2_120:painter" scale="4" />

The Brush is used to recolor hardened Construction Foam Walls.

## How to Use

- Hold a dye in the offhand and right-click with the Brush to load that color. The dye is consumed only when changing to a new color.
- Right-click a hardened Construction Foam Wall to paint it. Uncured Foam, Reinforced Foam, Reinforced Stone, and other blocks cannot be painted.
- The Brush has 32 uses. When its colored durability is exhausted, it resets to an uncolored Brush instead of breaking.
- All 16 dye colors are supported, and the Brush name changes to show its current color.
