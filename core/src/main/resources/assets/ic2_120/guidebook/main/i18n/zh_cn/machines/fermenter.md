# 发酵机

<BlockImage id="ic2_120:fermenter" p:facing="north" p:active="true" scale="4" />

发酵机通过发酵过程将生物质（Biomass）转化为生物燃料（Biofuel）。它拥有 8,000 mB（8 桶）的内部流体储罐，将生物质加工成更高效的燃料。

## 输出

- **流体储罐：** 8,000 mB
- **转化：** 生物质 → 生物燃料

## 槽位

- 左上方：流体容器输入（装有生物质）
- 右上方：流体容器输出（装有生物燃料）
- 下方：排出槽位
