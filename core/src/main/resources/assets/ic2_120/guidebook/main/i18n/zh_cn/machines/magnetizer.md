# 磁化机

<BlockImage id="ic2_120:magnetizer" p:facing="north" scale="4" />

磁化机可以产生磁场，吸引或排斥范围内的含铁物品和实体，消耗 EU 来维持效果。

## 工作方式

通电后，磁化机会产生一个磁场，与含铁物品、穿着铁盔甲的实体以及其他金属物体相互作用。磁场方向（吸引或排斥）可以配置。

## 使用

- **吸引模式**：将附近的铁质物品、穿着铁盔甲的实体和金属物体拉向磁化机。
- **排斥模式**：将附近的金属物体推离磁化机。

磁化机可用于分拣金属物品、构建物品运输系统，或通过排斥穿戴盔甲的实体来防御区域。
