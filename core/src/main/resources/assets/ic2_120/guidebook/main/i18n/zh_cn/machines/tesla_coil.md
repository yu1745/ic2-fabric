# 特斯拉线圈

<BlockImage id="ic2_120:tesla_coil" p:facing="north" scale="4" />

特斯拉线圈是一种电力防御装置，它会向附近的实体和怪物释放闪电进行攻击，接触时造成伤害。

## 能量与存储

- **EU 存储**：5,000 EU

## 工作方式

通电并充能后，特斯拉线圈会检测范围内的实体并对其释放闪电攻击，造成大量伤害。每次攻击消耗内部缓冲区中的 EU。只要有 EU 存储，线圈就会持续攻击。

## 使用

在基地周围放置特斯拉线圈以抵御敌对怪物。它们也可用于怪物农场——将线圈放置在刷怪区域附近，自动消灭怪物并收集掉落物。确保稳定的 EU 供应以保持线圈持续运行。
