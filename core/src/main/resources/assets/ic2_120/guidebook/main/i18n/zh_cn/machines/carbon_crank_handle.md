# 碳纤维摇杆

碳纤维摇杆是用于手摇动能发生机的最高级摇杆。安装后可使机器在摇动时产生 512 KU/t。

这是最高等级的摇杆，拥有最大的动能输出，适合大规模工业化生产中的极限动能需求。
