# Recycler

<BlockImage id="ic2_120:recycler" p:facing="north" p:active="true" scale="4" />

The Recycler breaks down any item or block into Scrap with a 12.5% chance per operation. It consumes EU and processes items continuously. The resulting Scrap can be used in the Matter Generator to boost UU-Matter production.

## Output

- **EU Storage:** 416 EU
- **Tier:** 1
- **Energy Usage:** ~8 EU/t per operation
- **Scrap Chance:** 12.5% per item

## Slots

- Top slot: input items to recycle
- Bottom slot: scrap output
