# 泵附件

<BlockImage id="ic2_120:bronze_pump_attachment" scale="4" />

泵附件是带方向的管道方块，会把其中一面变成被动抽取口。前方贴片必须贴住要抽取的储罐或机器，其他侧面则接回管道网络。

## 对比

| 类型 | 抽取速率 |
|------|---------|
| 青铜泵附件 | 2.4 B/s |
| 碳纤维泵附件 | 4.8 B/s |

## 使用方式

- 放置时让附件正面朝向源储存方块。它的正面不会连接普通管道。
- 被贴住的储存方块必须支持流体抽取，管网才会把该附件视为流体提供端。
- 不需要 EU。
- 右键泵附件可打开过滤界面。用装有流体的容器点击幽灵槽，可把附件锁定为该流体；右键或空手点击可清除过滤。
- 同一个管网里如果多个泵附件同时提供不同流体，网络会停滞。搬运多种流体时请使用过滤器或分开管网。

## 管网提示

泵附件适合储罐到储罐的物流，例如从发酵机抽出生物燃料送往发电机，或把冷凝机产出的蒸馏水导出。需要移除世界中的流体源方块时，则使用耗电的泵。
