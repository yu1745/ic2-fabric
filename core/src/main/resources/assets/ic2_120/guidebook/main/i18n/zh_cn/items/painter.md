# 刷子

<ItemImage id="ic2_120:painter" scale="4" />

刷子用于为固化后的建筑泡沫墙染色。

## 使用方式

- 将染料放在副手，手持刷子右键即可读取颜色。只有切换到新颜色时才会消耗染料。
- 右键固化后的建筑泡沫墙即可染色。未固化建筑泡沫、强化建筑泡沫、防爆石和其他方块不能染色。
- 刷子有 32 次使用耐久。带颜色的刷子耐久耗尽后会恢复为未染色的刷子，而不会损坏消失。
- 支持全部 16 种染料颜色，刷子名称会显示当前颜色。
