# 采矿镭射枪

<ItemImage id="ic2_120:mining_laser" scale="4" />

<ItemLink id="ic2_120:mining_laser" /> 是一种高耗电、远程的采矿工具，可发射一道可见的 EU 供能光束。它能破坏方块、攻击实体，并可根据发射模式的不同就地烧制矿石或开出隧道。它是 <ItemLink id="ic2_120:energy_crystal" /> 或 <ItemLink id="ic2_120:lapotron_crystal" /> 典型的终期消耗者。

本页面只是一个简明概述。完整的模式说明、弹射体规则、随机点火机制与实体伤害缩放等请参考专门的 [采矿镭射枪指南](../guides/mining_laser.md)。

## 属性

采矿镭射枪是一件手持物品，内置大型 EU 缓冲。它从玩家物品栏内任意兼容的电池中充电，并在右键时以脉冲形式释放缓冲中的电量。在低功率模式下，一满缓冲可支撑多次射击；高功率模式则次数更少。

| 属性 | 取值 |
|----------|------|
| 等级 | 3（能量水晶） |
| 内置缓冲 | 可配置，默认较大，在 `Ic2Config` 中设置 |
| 每次射击 EU | 随模式而异（详见专门指南） |
| 充电源 | 物品栏内任意能量水晶等级或更高级别的电池 |
| 操作 | 右键发射；按住 **模式切换** + 右键循环切换模式 |

默认值可在模组的 `Ic2Config` 中调整；具体缓冲容量请查看对应版本配置文件。

## 模式一览

采矿镭射枪具有多种发射模式。完整的消耗表、射程与对实体伤害请见专门的指南页面；下表仅用于说明该工具能够做什么。

- **低聚焦**——低耗、超短距模式，适合精细的单格挖掘。
- **采矿**——标准中距模式，一道光束会一格接一格连续贯穿多个方块，直到本发可挖距离耗尽。
- **远距**——快速、长距的弹射体，用于远程狙击方块或实体。
- **超级热线**——高功率模式，可就地烧制可烧炼的矿石（例如铁矿 → 铁锭）。因其处理目标方块的方式，也被称作「岩浆模式」。
- **散射**——单次右键发射一排扇形光束，适合清理较大面积。
- **爆破**——在命中点引爆一个 TNT 级的爆炸，对范围内任何实体造成高额真实伤害。
- **3×3**——向前发射一面 3×3 的光束墙，适合开凿隧道。

完整的每模式消耗、射程与对实体伤害表见 [采矿镭射枪指南](../guides/mining_laser.md)。

## 网络与权威

采矿镭射枪是服务器权威的工具。`MiningLaserServerSuppress` 规则用于确保开火决定由服务器做出，并拒绝任何在客户端侧刷发射的尝试。实际效果很简单：只有服务器判定缓冲足够时才会开火，无论客户端如何预测。

## 使用方式

1. 在物品栏中携带一颗已充电的 <ItemLink id="ic2_120:lapotron_crystal" />（或更高级的电源）。
2. 将 <ItemLink id="ic2_120:mining_laser" /> 拿在主手。行动条会短暂显示当前模式。
3. 右键发射，射击从镭射枪的内置缓冲中消耗 EU。
4. 切换模式时，按住 **模式切换** 键并右键。该键位由其他 IC2 工具共享，默认绑定位于模组按键包的 **选项 → 控制** 中。
5. 补充电量时，把镭射枪放入 BatBox、CESU、MFE 或 MFSU 的电池槽。详见 [储电设备](../machines/energy_storage.md)。
