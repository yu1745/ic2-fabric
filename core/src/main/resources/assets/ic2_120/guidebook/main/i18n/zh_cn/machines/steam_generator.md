# 蒸汽发生器

<BlockImage id="ic2_120:steam_generator" p:facing="north" p:active="true" scale="4" />

蒸汽发生器（锅炉）消耗热量（HU）和水来产生蒸汽或过热蒸汽。它从相邻热源接收热量，通过流体管道或容器接收水。1 桶水可产生 100 桶蒸汽（100 倍膨胀）。

当系统温度达到 374°C 时，蒸汽发生器开始产生过热蒸汽而非普通蒸汽。最高安全运行温度为 500°C - 超过此温度会爆炸。使用非蒸馏水时，机器会逐渐积累水垢，最终导致停摆。

### 控制

蒸汽发生器 GUI 提供进水速率（mB/t）和压力两个可调参数，影响蒸汽产量和温度。进水速率范围为 0 至 1000 mB/t，压力范围为 0 至 300 bar。

## 输出

- **蒸汽输出**：每 1 mB 水产生 100 mB 蒸汽（100 倍膨胀）
- **水罐**：10 桶
- **蒸汽罐**：100,000 mB
- **最大热输入**：1,200 HU/t
- **等级**：2
- **最高温度**：500°C（超过会爆炸）

### 蒸汽类型

| 类型 | 温度 | 说明 |
|------|------|------|
| 普通蒸汽 | 374°C 以下 | 常规蒸汽 |
| 过热蒸汽 | 374°C 以上 | 更高效率 |

## 槽位

蒸汽发生器没有物品槽。所有流体通过连接的流体管道和相邻热源处理。它通过流体管道连接从所有面接收水并输出蒸汽。
