# Fermenter

<BlockImage id="ic2_120:fermenter" p:facing="north" p:active="true" scale="4" />

The Fermenter converts Biomass into Biofuel through a fermentation process. It has an internal 8 bucket (8,000 mB) tank for fluid storage and processes biomass into a more efficient fuel source.

## Output

- **Fluid Tank:** 8,000 mB
- **Conversion:** Biomass → Biofuel

## Slots

- Top left: fluid input cell
- Top right: fluid output cell
- Bottom: discharge slot
