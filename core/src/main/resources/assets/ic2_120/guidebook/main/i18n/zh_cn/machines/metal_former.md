# 金属成型机

<BlockImage id="ic2_120:metal_former" p:facing="north" p:active="true" scale="4" />

金属成型机通过三种不同模式将金属加工成各种形态：轧制（生产板材）、挤压（生产线缆和棒材）和切割（生产导线和外壳）。可在机器 GUI 中切换模式。

## 输出

- **等级：** 1
- **能量消耗：** 每次操作约 20 EU/t

## 槽位

- 左侧槽位：输入
- 右侧槽位：输出
