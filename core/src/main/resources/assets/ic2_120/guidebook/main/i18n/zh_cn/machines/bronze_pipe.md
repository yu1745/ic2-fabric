# 青铜管道

<BlockImage id="ic2_120:bronze_pipe_tiny" scale="4" />

青铜管道用于在储罐和机器之间输送流体。它们有四种尺寸，具有不同的吞吐能力。青铜管道会自动连接相邻的储罐、机器和其他管道。

## 尺寸对比

| 尺寸 | 吞吐量 | 用途 |
|------|--------|------|
| 微型 | 0.5 B/t (最大 500 mB/t) | 低流量精确输送 |
| 小型 | 1 B/t (最大 1,000 mB/t) | 通用输送 |
| 中型 | 2 B/t (最大 2,000 mB/t) | 高流量输送 |
| 大型 | 4 B/t (最大 4,000 mB/t) | 大量流体输送 |

## 使用方式

- 管道会自动连接相邻的储罐、机器和其他管道。
- 流体会通过连接的管道从源（储罐、泵）流向目的地。
- 需要安装泵附件才能从储罐和机器中自动抽取流体。
- 使用扳手右键点击管道可配置连接面。
- 拆卸时管道会保留其中的流体。
