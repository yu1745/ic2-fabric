# 固体装罐机

<BlockImage id="ic2_120:solid_canner" p:facing="north" p:active="true" scale="4" />

固体装罐机用于将固体物品装入或清空容器。它可用于填充燃料罐、装填弹药以及其他固体装罐操作。

## 输出

- **EU 存储：** 416 EU
- **等级：** 1

## 槽位

- 上方行（3 个槽位）：容器 / 燃料 / 填充物槽位
- 下方行（3 个槽位）：输出槽位
