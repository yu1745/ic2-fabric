# 作物收割机

<BlockImage id="ic2_120:crop_harvester" p:facing="north" scale="4" />

作物收割机会自动收获以机器为中心的 **3x3x3** 范围内完全成熟的 IC2 作物，收集掉落物并保留作物植株以便继续生长。

## 能量与存储

- **EU 存储**：10,000 EU
- **等级**：1 级（LV，32 EU/packet）

## 工作方式

作物收割机会扫描其 3x3x3 范围内的成熟 IC2 作物。发现成熟作物后，它会进行收割，将所有掉落物收集到输出栏位中。作物植株会保留在原位，无需重新种植即可继续生长。

## 使用

将作物收割机放置在作物农场的中心位置。提供 EU 电力并确保输出槽有足够空间存放收获物。收割机会自动收取成熟作物，大幅减少大规模农耕的人工操作。
