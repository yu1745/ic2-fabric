# 创造模式发电机

<BlockImage id="ic2_120:creative_generator" p:facing="north" p:active="true" scale="4" />

创造模式发电机为测试和创造建筑提供无限 EU。它无需任何燃料即可直接输出 512 EU/t，六面均可输出。

该方块仅限创造模式使用，适用于机器测试、电网验证以及需要无限能源的建筑方案。

## 输出

- **EU 输出**：512 EU/t（无限能源）
- **缓存**：无（直通）
- **等级**：1
- **仅创造模式**：生存模式下无法获得

## 配方

*生存模式下无法获得。*
