# Pump

<BlockImage id="ic2_120:pump" p:facing="north" p:active="true" scale="4" />

The Pump collects fluids from the world and stores them in its internal tank. When placed adjacent to a fluid source, it will automatically pump the fluid into its tank, which can then be extracted using fluid cells or pipes.

## Output

- **EU Storage:** 800 EU
- **Tier:** 1
- **Energy Usage:** ~16 EU/t while pumping

## Slots

- Top slot: fluid container (for collecting pumped fluid)
- Bottom slot: empty fluid container input
