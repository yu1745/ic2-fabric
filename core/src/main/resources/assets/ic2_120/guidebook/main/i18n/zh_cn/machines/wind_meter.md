# 风力计

风力计是一种手持工具，右键点击时可测量当前的风力。它会显示玩家所在位置的有效风速，综合考虑了高度风力分布、阵风因子和天气倍率。

使用风力计可以为风力动能发生机和风力发电机寻找最佳安装位置。
