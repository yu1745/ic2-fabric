# Advanced Miner

<BlockImage id="ic2_120:advanced_miner" p:facing="north" scale="4" />

The Advanced Miner is an upgraded version of the basic Miner with faster operation, a larger mining range, and additional features for more efficient automated mining.

## Improvements over the Basic Miner

- **Faster Mining**: Operates at a higher speed than the basic Miner.
- **Larger Range**: Supports a wider mining radius with improved scanner support.
- **Enhanced Scanner Support**: Better integration with OD and OV scanners for precise area control.

## Operation

Like the basic Miner, the Advanced Miner drills downward using mining pipes and excavates blocks within its configured radius. Its advanced mechanism allows it to process blocks more quickly and cover a larger area per operation.

## Scanners

- **OD Scanner**: Configures the vertical and horizontal mining radius.
- **OV Scanner**: Expands the effective scanning range for wider area coverage.

## Requirements

- **EU**: Power for the mining mechanism (higher consumption than basic Miner).
- **Mining Pipes**: Consumed as the miner extends deeper.
- **Scanners** (optional): For configuring the mining area.

## Usage

The Advanced Miner is ideal for large-scale mining operations. Set it up above a rich ore vein, provide ample EU power and mining pipes, and let it handle the excavation while you focus on other tasks. Its larger range and faster speed make it significantly more productive than the basic Miner.
