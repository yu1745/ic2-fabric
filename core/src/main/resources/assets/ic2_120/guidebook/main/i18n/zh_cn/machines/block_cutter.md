# 方块切割机

<BlockImage id="ic2_120:block_cutter" p:facing="north" p:active="true" scale="4" />

方块切割机将方块切割成更小或更精炼的形态。它需要钻石刀片作为工具组件，消耗 EU 来运行，可产出各种建筑材料的切割版本。

## 输出

- **EU 存储：** 3,600 EU
- **能量消耗：** 因操作而异
- **消耗品：** 钻石刀片

## 槽位

- 上方槽位：钻石刀片
- 中间槽位：输入方块
- 下方槽位：输出
