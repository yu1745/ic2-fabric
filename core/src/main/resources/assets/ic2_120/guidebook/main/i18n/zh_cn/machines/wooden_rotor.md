# 木制转子

<ItemImage id="ic2_120:wooden_rotor" scale="4" />

木制转子是用于风力动能发生机和水力动能发生机的基础转子。

- **风力**：半径 2，倍率 1×
- **水力**：半径 1.0，倍率 1×

这是最低等级的转子，功率输出最小，旋转半径也最小。
