# 高级采矿机

<BlockImage id="ic2_120:advanced_miner" p:facing="north" scale="4" />

高级采矿机是基础采矿机的升级版本，具有更快的运行速度、更大的采矿范围和更高的自动化效率。

## 相较于基础采矿机的改进

- **更快的采矿速度**：运行速度比基础采矿机更高。
- **更大的范围**：支持更宽的采矿半径，并改进了扫描仪兼容性。
- **增强的扫描仪支持**：与 OD 和 OV 扫描仪更好地集成，实现精确的区域控制。

## 工作方式

与基础采矿机类似，高级采矿机使用采矿管道向下钻孔，并在配置的半径内开采方块。其先进的机械结构使其能够更快地处理方块，每次操作覆盖更大的区域。

## 扫描仪

- **OD 扫描仪**：配置垂直和水平采矿半径。
- **OV 扫描仪**：扩展有效扫描范围，覆盖更广的区域。

## 需求

- **EU**：为采矿机构提供电力（消耗高于基础采矿机）。
- **采矿管道**：随着采矿机向下延伸而消耗。
- **扫描仪**（可选）：用于配置采矿区域。

## 使用

高级采矿机非常适合大规模采矿作业。将其放置在富矿脉上方，提供充足的 EU 电力和采矿管道，让它自动完成挖掘工作，您可以专注于其他任务。更大的范围和更快的速度使其效率显著高于基础采矿机。
