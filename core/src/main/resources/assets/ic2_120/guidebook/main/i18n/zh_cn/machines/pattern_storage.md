# 模式存储机

<BlockImage id="ic2_120:pattern_storage" p:facing="north" scale="4" />

模式存储机用于存储扫描后的 UU 物质模式，充当模式扫描机和复制机之间的桥梁。它必须与扫描机和复制机相邻放置，以便在它们之间传输模式数据。

## 工作方式

模式存储机保存由模式扫描机创建的数字模式。模式存储后，任何相邻的复制机都可以访问这些模式，开始使用 UU 物质复制物品。

## 放置

- 将模式存储机放在**模式扫描机**旁边，以接收扫描完成的模式。
- 将模式存储机放在**复制机**旁边，使这些模式可用于复制。
- 模式存储机可以同时与两者相邻，充当中央集线器。

## 使用

1. 使用模式扫描机扫描物品——模式会自动传输到连接的模式存储机。
2. 将复制机连接到模式存储机以访问存储的模式。
3. 在复制机中选择所需的模式，开始复制。

可以使用多个模式存储机来存储大量物品的模式目录。
