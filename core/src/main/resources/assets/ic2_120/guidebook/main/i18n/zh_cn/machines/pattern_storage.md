# 模式存储机

<BlockImage id="ic2_120:pattern_storage" p:facing="north" scale="4" />

模式存储机用于存储扫描后的 UU 物质模式，充当模式扫描机和复制机之间的桥梁。它必须与扫描机和复制机相邻放置，以便在它们之间传输模式数据。

## 工作方式

模式存储机会保存一组已扫描模板，并记录一个当前选中的模板。模式扫描机可以把完成的扫描写入其中，复制机则会读取唯一相邻模式存储机中当前选中的模板。

写入已有模板时，会更新该条目并选中它。删除当前模板后，选择会移动到剩余有效条目；如果列表为空则清空选择。

## 模式水晶

唯一的物品槽接受模式水晶。模式存储机可以从水晶导入模板，也可以把当前选中的模板导出到水晶。这样可以在不搬机器的情况下转移模板。

## 放置规则

- 保存扫描结果时，把模式存储机放在模式扫描机旁边。
- 复制物品时，把模式存储机放在复制机旁边。
- 不要让同一台扫描机或复制机旁边同时接触多个模式存储机；这些机器会查找唯一相邻存储，目标不唯一时会停止工作。

模式存储机不消耗 EU。它唯一的自动化接口是模式水晶槽，可插入和抽出模式水晶。
