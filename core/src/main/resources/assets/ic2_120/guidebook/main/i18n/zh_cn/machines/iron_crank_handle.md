# 铁制摇杆

<ItemImage id="ic2_120:iron_crank_handle" scale="4" />

铁制摇杆是用于手摇动能发生机的中级摇杆。安装后可使机器在摇动时产生 128 KU/t。

与木制摇杆相比，铁制摇杆的输出功率翻倍，适合中期游戏的动能需求。
