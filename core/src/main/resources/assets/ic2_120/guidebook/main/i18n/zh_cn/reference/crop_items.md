# 作物物品与饮品

作物系统除了农作物本身，还包含种子袋、扫描、除草和饮品链。

## 作物工具

- **作物架**：放在耕地上作为 IC2 作物的基础。
- **作物种子袋**：保存作物类型、成长、产量、抗性等属性。
- **种子扫描仪**：分析种子袋属性。
- **除草铲**：清理作物架上的杂草。
- **除草剂**：消耗内部除草剂容量抑制杂草。
- **肥料**：提高作物护理效率。

## 收获物

作物可产出啤酒花、咖啡豆、大地疣、杂草、蛤蛤粉、面粉等物品。具体作物收获见：[作物收获产出参考](crop_harvest.md)。

## 饮品

- 石杯用于承载饮品。
- 咖啡豆可加工为咖啡粉，并进一步制作咖啡、冷咖啡、黑咖啡。
- 作物饮品链适合与自动农场和加工机器配合。

玩法入口：[作物系统指南](../guides/crop_player_guide.md)
