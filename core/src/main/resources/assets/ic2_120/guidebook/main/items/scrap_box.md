# Scrap Box

<ItemImage id="ic2_120:scrap_box" scale="4" />

The Scrap Box is a consumable item that produces a random item when right-clicked. It is crafted from <ItemLink id="ic2_120:scrap" />, which is the output of the <ItemLink id="ic2_120:recycler" />. The Scrap Box is a fun way to turn excess recyclable materials into potentially valuable items.

## How to Use

- Right-click the Scrap Box in hand to consume it and receive a random item.
- The output pool includes many vanilla and IC2 items — anything from dirt and cobblestone to diamonds and IC2 resources.
- There is no guarantee of getting a valuable item; the Scrap Box is essentially a gamble.
