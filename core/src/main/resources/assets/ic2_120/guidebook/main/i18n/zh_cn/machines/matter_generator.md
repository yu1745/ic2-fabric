# 物质生成机

<BlockImage id="ic2_120:matter_generator" p:facing="north" scale="4" />

物质生成机消耗大量 EU 来生产 **UU 物质**，供复制机复制物品使用。它只有在收到红石信号且内部储罐仍有空间时才会运行。

## 工作方式

机器以 1 mB 为单位把 EU 转化为 UU 物质。

- **能量等级：** 3
- **EU 存储：** 4,000,000 EU
- **UU 物质储罐：** 10,000 mB
- **无废料：** 每 mB 1,000,000 EU
- **有废料：** 每 mB 166,667 EU + 34 个废料

废料会随着进度推进而消耗，因此少量废料也能生效，但持续保持加速需要稳定的回收机供料。

## 槽位

- **废料槽：** 只接受废料
- **容器输入槽：** 空单元、空流体单元或桶
- **容器输出槽：** UU 物质单元、装有 UU 物质的流体单元或 UU 物质桶
- **升级槽：** 4 个槽位

储罐对管道只输出不输入。输入槽中的空容器会在储罐至少有 1 桶 UU 物质时被灌装。

## 自动化与升级

物品自动化可以插入废料、空容器和支持的升级。废料槽、容器输入槽和容器输出槽都可抽出。

支持变压、储能、弹出、拉取、流体弹出和流体拉取升级。不接受超频升级。流体弹出升级可把 UU 物质推送到储罐或复制机；由于内部储罐拒绝外部输入，流体拉取升级通常没有实际输入用途。

## 复制链路

常见链路是：回收机 -> 物质生成机 -> 复制机。用回收机持续供应废料，以红石信号启动物质生成机，再把 UU 物质输送到复制机或储罐。
