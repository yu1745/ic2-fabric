# 物质生成机

<BlockImage id="ic2_120:matter_generator" p:facing="north" scale="4" />

物质生成机消耗大量 EU 来生产 **UU 物质**，UU 物质是用于复制任何材料的基础物质。投入废料可以降低每毫桶的 EU 消耗，从而加速生产过程。

## 工作方式

物质生成机将 EU 转换为 UU 物质流体。没有废料时，转换速度慢且能耗极高：

- **无废料**：每 mB UU 物质的 EU 成本很高
- **有废料**：废料降低每 mB 所需的 EU，显著加快生产速度

## 输入

- **EU**（主要输入）：运行时持续消耗
- **废料**（可选）：放入输入槽以提高生产速度

## 输出

机器产出液态 **UU 物质**，存储在内部储罐中，可通过流体管道或桶提取。UU 物质是复制机复制物品的关键原料。

## 使用

向物质生成机持续供应 EU。为获得最佳效率，同时供应废料——废料可通过回收机处理常见物品获得。产出的 UU 物质可直接输送到相邻的复制机，或存储在储罐中供后续使用。
