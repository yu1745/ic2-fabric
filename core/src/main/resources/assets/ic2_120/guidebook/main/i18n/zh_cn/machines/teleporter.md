# 传送机

<BlockImage id="ic2_120:teleporter" p:facing="north" scale="4" />

传送机可以在两个已配对的传送机之间瞬间传送玩家和实体。EU 消耗随传送距离增加而增加。

## 能量

- **等级**：4 级（EV，2048 EU/packet）

## 工作方式

设置传送的步骤：

1. 在源位置和目标位置各放置一台传送机。
2. 为两台传送机提供充足的 EU 电力（4 级 / EV）。
3. 传送消耗的 EU 根据两台传送机之间的距离计算。
4. 走上传送机即可激活——站在传送台上的实体会被传送。

## EU 消耗

传送所需的能量取决于两台传送机之间的欧几里得距离。距离越远，消耗的 EU 越多。请确保您的电力基础设施能够满足需求，尤其是远距离传送时。

## 使用

传送机非常适合在基地、采矿前哨站和远程设施之间快速旅行。通过配对多台传送机来连接多个地点。更高的 EU 存储和发电能力支持更远距离的传送。
