# 牲畜屠宰机

<BlockImage id="ic2_120:animal_slaughterer" p:facing="north" scale="4" />

牲畜屠宰机会自动处理以机器为中心的 **3x3x3** 范围内完全成长的成年动物，收集肉类、皮革、羊毛等动物掉落物。

## 能量与存储

- **EU 存储**：10,000 EU
- **等级**：1 级（LV，32 EU/packet）

## 工作方式

牲畜屠宰机会扫描其 3x3x3 范围内的成年动物。检测到成熟动物后，会对其进行处理并将掉落物收集到输出栏位中。幼年动物不会受到影响。

## 使用

将牲畜屠宰机放置在畜栏旁边。提供 EU 电力并确保输出槽有足够空间。与牲畜监管机配合使用可实现全自动繁殖-屠宰循环：监管机负责繁殖动物，屠宰机在动物成年后进行收获。
