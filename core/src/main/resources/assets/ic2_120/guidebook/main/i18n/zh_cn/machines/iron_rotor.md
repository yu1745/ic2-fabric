# 铁制转子

<ItemImage id="ic2_120:iron_rotor" scale="4" />

铁制转子是用于风力动能发生机和水力动能发生机的中级转子。

- **风力**：半径 3，倍率 2×
- **水力**：半径 1.5，倍率 2×

与木制转子相比，铁制转子拥有更大的旋转半径和更高的输出倍率，显著提升能量采集效率。
