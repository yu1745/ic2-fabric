# 材料与矿石

## 世界资源

- 矿石：锡、铅、铀、泥炭，以及对应深层矿石。
- 稀有资源：铱矿石、铱碎片、强化铱板，是后期储能和核电组件的关键材料。
- 橡胶：橡胶树提供粘性树脂，树脂可加工为橡胶。

## 金属与基础材料

| 材料 | 常见形态 | 用途 |
|------|----------|------|
| 铜 | 粉、板、外壳、导线 | 早期电路、热交换、导线 |
| 锡 | 锭、粉、板、外壳、单元 | 电池、单元、冷却组件 |
| 青铜 | 锭、粉、板、外壳、工具 | 早期工具、储罐、管道 |
| 铅 | 锭、粉、板、外壳 | 核电、屏蔽、部分电池 |
| 银 | 锭、粉 | 玻璃纤维导线等高级配方 |
| 钢 | 锭、板、外壳 | 强化结构、工具、机器 |
| 铀 | 矿、锭、粉、燃料 | 核燃料链 |

## 加工链

- 原矿可进入打粉、洗矿、热离心等链路，提高产出或得到副产物。
- 粉碎矿石可进一步洗成纯净粉碎矿石。
- 粉、锭、板、外壳、致密板是多数机器和工具的中间件。
- 锻造锤把锭加工为板、把板加工为外壳。
- 板材切割剪刀把板材切割为导线，或剥离绝缘导线。

## 碳与高级材料

- 煤粉、煤球、压缩煤、煤块可进入碳材料链。
- 粗制碳网、粗制碳板、碳板用于高级工具、装甲和转子。
- 高级合金、工业钻石、能量水晶粉、铱板用于后期机器与装备。
