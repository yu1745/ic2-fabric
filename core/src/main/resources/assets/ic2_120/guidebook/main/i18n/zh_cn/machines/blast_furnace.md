# 高炉

<BlockImage id="ic2_120:blast_furnace" p:facing="north" p:active="true" scale="4" />

高炉用于将铁质材料冶炼成钢。它不直接使用 EU：必须从受热面输入热量（HU），并在配方推进时持续消耗压缩空气。

## 热量与温度

受热面就是机器朝向的一面，因此需要把供热机器贴在这一面，并让两台机器相互对准。高炉内部有 HU 缓冲，可持续接收热量。

- 温度范围：**0-1700 C**
- 开始工作温度：**1401 C**

炉子始终在散热，散热量（维持当前温度所需的 HU/t）随温度线性增长：

- 0-1401 C：散热从 **0 升到 50 HU/t**
- 1402-1700 C：散热从 **50 升到 100 HU/t**

每 tick 的净热量 = 输入 HU − 散热。净热量为正时累积并升温，为负时降温，恰好为零则温度稳定。**最终稳定温度完全由 HU 输入决定**：

- **50 HU/t** → 稳定在 **1401 C**（刚够开始工作）
- **100 HU/t** → 稳定在 **1700 C**
- 中间线性插值，例如 75 HU/t → 约 1550 C

升温速度正比于净 HU：HU 输入越高，升温越快。工作时温度仍然会根据 HU 输入正常升降，不再冻结。HU 不足导致温度降到 1401 C 以下时，炼钢进度会被清空。

温度越高越快：每生产 1 个钢锭大约需要 **1401 C 时 10000 tick**、**1500 C 时 8400 tick**、**1600 C 时 6000 tick**、**1700 C 时 4000 tick**。

## 空气与配方

压缩空气储存在内部 **8 桶 / 8000 mB** 储罐中。储罐可以通过 Fabric 流体管道输入压缩空气，也可以由空气输入槽补充。放入压缩空气单元，或其他装有整桶压缩空气的流体单元/容器时，会加入 1 桶压缩空气，并在适用时向空单元输出槽返还空单元。

压缩空气消耗会随温度升高而降低：

- 1401 C：每个钢锭约 **6000 mB**
- 1500 C：每个钢锭约 **5200 mB**
- 1600 C：每个钢锭约 **4700 mB**
- 1700 C：每个钢锭约 **4200 mB**

每个有效输入都会产出 **1 个钢锭** 和 **1 个炉渣**：

- 铁粉
- 粉碎铁矿石
- 纯净粉碎铁矿石
- 铁锭
- 铁矿石
- 深板岩铁矿石
- 粗铁

## 槽位

- **输入槽：** 高炉配方使用的铁质材料
- **空气输入槽：** 压缩空气单元或兼容的压缩空气容器
- **钢输出槽：** 钢锭
- **炉渣输出槽：** 炉渣
- **空单元输出槽：** 由空气单元返还的空单元
- **升级槽：** 2 个槽位，可放入物品拉取升级和弹出升级

## 自动化

物品自动化会把配方输入送入输入槽，把压缩空气单元送入空气输入槽。钢、炉渣和空单元输出槽可以被抽出。拉取升级会拉取配方输入或空气单元；弹出升级会弹出三个输出槽中的物品。

流体自动化可以从除受热面以外的任意侧向内部储罐输入压缩空气。该储罐只接受输入，不会把压缩空气重新输出到管道。

实用搭建方式是先把高炉加热到工作温度，在处理期间保持供热，持续补充压缩空气，并为钢锭和炉渣都留出输出空间。炉渣输出满时，后续钢锭生产也会停止。
