# 高炉

<BlockImage id="ic2_120:blast_furnace" p:facing="north" p:active="true" scale="4" />

高炉利用 IC2 热力网络的热量（HU）在高温下冶炼材料。它的主要用途是通过铁粉和煤粉的组合来生产钢。

## 输出

- **能量来源：** 热量（HU）
- **主要配方：** 铁粉 + 煤粉 → 钢

## 槽位

- 左上方：流体输入
- 右上方：流体输出
- 底部行：物品输入/输出
