# Block Cutting Machine

<BlockImage id="ic2_120:block_cutter" p:facing="north" p:active="true" scale="4" />

The Block Cutting Machine cuts blocks into smaller or more refined forms. It requires a diamond blade as a tool component and consumes EU to operate, producing cut versions of various building materials.

## Output

- **EU Storage:** 3,600 EU
- **Energy Usage:** varies per operation
- **Consumable:** Diamond Blade

## Slots

- Top slot: diamond blade
- Middle slot: input block
- Bottom slot: output
