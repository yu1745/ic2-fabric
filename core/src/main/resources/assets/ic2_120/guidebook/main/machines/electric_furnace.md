# Electric Furnace

<BlockImage id="ic2_120:electric_furnace" p:facing="north" p:active="true" scale="4" />

The Electric Furnace uses EU to smelt items instead of consuming solid fuel. It functions identically to a regular furnace but is powered by the IC2 energy network, making it ideal for automated setups.

## Output

- **EU Storage:** 416 EU
- **Tier:** 1
- **Energy Usage:** ~20 EU/t per operation

## Slots

- Top slot: input item to smelt
- Bottom slot: output
- Right slot: chargeable battery or electric tool
