# Steel Rotor

The Steel Rotor is a high-tier rotor used in Wind Kinetic Generators and Water Kinetic Generators.

- **Wind**: Radius 4, Multiplier 3&times;
- **Water**: Radius 2.0, Multiplier 3&times;

It provides excellent power generation capabilities, significantly outperforming lower-tier rotors in both radius and output multiplier.
