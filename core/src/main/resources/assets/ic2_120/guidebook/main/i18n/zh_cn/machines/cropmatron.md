# 作物监管机

<BlockImage id="ic2_120:cropmatron" p:facing="north" scale="4" />

作物监管机是一种自动化农耕机器，负责照看以机器为中心的 **3x3x3** 范围内的 IC2 作物。它可以自动浇水、除杂草和施肥。

## 能量与存储

- **EU 存储**：10,000 EU
- **等级**：1 级（LV，32 EU/packet）
- **水箱**：8 桶（用于浇水和混合除草剂）

## 功能

作物监管机执行三项关键的作物维护任务：

- **浇水**：自动浇灌范围内的作物，消耗内部水箱中的水。
- **除草**：喷洒除草剂清除作物田中的杂草。需要消耗水箱中的水。
- **施肥**：施加肥料以促进作物生长。需要在输入槽中提供肥料。

## 使用

将作物监管机放置在作物农场的中心位置。它会自动维护周围 3x3x3 区域内的作物。为机器提供 EU 电力，并通过流体管道或水桶保持水箱充满。在输入槽中放入肥料即可自动施肥。
