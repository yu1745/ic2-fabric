# 核反应堆（EU 模式）

<BlockImage id="ic2_120:nuclear_reactor" scale="4" />

核反应堆是一种强大的后期 EU 发电设备，通过受控核裂变产生能量。它构成一个多方块结构，需要仔细的热量管理才能安全运行。

本页介绍 **EU 模式**（干式，输出 EU）。如需流体冷却的**流体模式**（压力容器外壳、流体接口、输出 HU），见 [流体冷却反应堆](nuclear_reactor_fluid_mode.md)。

## 多方块结构

核反应堆多方块结构由以下组件构成：

- **核反应堆**：结构核心，内置 3x3 或 6x9（安装反应仓后）的元件网格，用于放置燃料棒和功能组件。
- **核反应仓**：安装在反应堆侧面（最多 6 个，每个面一个）。每个反应仓会扩展内部元件网格，允许放置更多燃料棒、冷却单元和热交换器。

**反应堆访问接口**、**反应堆流体接口**、**反应堆红石接口**和**核反应堆压力容器**详见 [流体冷却反应堆](nuclear_reactor_fluid_mode.md) 页面——这四种方块两种模式通用，但因它们定义了流体冷却流体模式的外壳，统一在那里说明。

## 发电

### 发电公式

```
基础脉冲 = 1 + 联数 / 2
相邻脉冲 = 相邻的燃料棒 + 反射板数量（每个贡献 1）
总脉冲 = (基础脉冲 + 相邻脉冲) × 联数
EU/t = 总脉冲 × 5
```

`联数` 为燃料棒的联数（单联=1，双联=2，四联=4）。每有一个正交相邻的燃料棒或中子反射板，脉冲数 +1。反应堆每秒（20 tick）处理一次燃料棒，每秒总量（`总脉冲 × 100`）均匀释放，折算为 `总脉冲 × 5` EU/t。

**注意：** 相邻脉冲按**占用的格子数**计，不按联数计。相邻的单联、双联或四联燃料棒都只贡献 **+1 脉冲**——旁边的四联燃料棒并不会 +4。

| 燃料棒类型 | 基础 EU/t（无相邻） |
|----------|---:|
| 单联铀燃料棒 | 5 EU/t |
| 双联铀燃料棒 | 20 EU/t |
| 四联铀燃料棒 | 60 EU/t |

### MOX 燃料棒

MOX 输出随反应堆温度提升（`热量 / 最大热量`）：

```
倍率 = 4 × (热量 / 最大热量) + 1
EU/t = 总脉冲 × 倍率 × 5
```

| 反应堆温度 | 倍率 | 单联 MOX EU/t（无相邻） |
|---:|---:|---:|
| 0 | ×1 | 5 |
| 2,500 | ×2 | 10 |
| 5,000 | ×3 | 15 |
| 7,500 | ×4 | 20 |
| 10,000 | ×5 | 25 |

## 热量管理

反应堆的基础热容量为 **10,000 HU**。若热量超过此上限，反应堆将发生灾难性核熔毁并爆炸。热量由燃料棒在运行过程中产生，必须通过以下方式管理：

- **冷却单元**：吸收相邻燃料棒的热量，并随时间缓慢消散。
- **热交换器**：在元件之间重新分配热量，平衡热负荷。
- **元件散热片**：直接从相邻元件散热。
- **超频散热片**：以更高效率散热。

### 反应堆隔板

在网格中安装**反应堆隔板**或**密封反应堆隔热板**可提升最大热容量，并降低爆炸威力。隔板的热容量加成为加法叠加，爆炸威力修正为乘法叠加：

| 组件 | 最大热量加成 | 爆炸威力 |
|------|---:|---:|
| 反应堆隔板 | +500 HU | ×0.9 |
| 密封反应堆隔热板 | +1,700 HU | ×0.99 |

### 温度阈值

| 反应堆温度 | 环境影响 |
|---:|---|
| &gt; 4,000 | 5×5×5 范围内的随机方块可能起火 |
| &gt; 5,000 | 5×5×5 范围内的随机水可能蒸发 |
| &gt; 7,000 | 7×7×7 范围内的实体受到核高温伤害 |
| &gt; 8,500 | 5×5×5 范围内的随机方块可能变为熔岩 |
| ≥ 10,000 | **核反应堆爆炸** |

穿戴防化服可免疫 7,000 以上的**核高温伤害**（这是高温伤害，不是辐射伤害）。

每种燃料棒、散热片、热交换器和反应堆隔板的完整数值，见 [核反应堆方块与组件](../reference/nuclear_components.md)。

## 输出

反应堆通过其核心方块输出 EU。使用 HV 或 EV 电缆将反应堆连接至电网。反应堆会自动将 EU 分配给连接的电缆、储能设备和机器。

## 常数

| 常数 | 数值 |
|------|------|
| 最大输出 | 8,192 EU/t（5 级） |
| 能量缓冲 | 1,000,000 EU |
| 反应堆温度上限 | 10,000 HU |
