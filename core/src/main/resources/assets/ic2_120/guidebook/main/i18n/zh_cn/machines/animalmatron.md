# 牲畜监管机

<BlockImage id="ic2_120:animalmatron" p:facing="north" scale="4" />

牲畜监管机是一种自动化畜牧管理机器，负责喂食和繁殖以机器为中心的 **3x3x3** 范围内的动物。

## 能量与存储

- **EU 存储**：10,000 EU
- **水箱**：8 桶

## 功能

- **喂食**：使用输入槽中的食物自动喂食范围内的动物。
- **繁殖**：动物进食后会进入求爱模式并自动繁殖，产生后代。
- **用水**：某些喂食操作需要消耗水。

## 使用

将牲畜监管机放置在畜栏附近。在输入槽中放入对应的动物食物（小麦喂牛、胡萝卜喂猪、种子喂鸡等）。确保机器有 EU 供电且水箱中有水。监管机会自动处理范围内的喂食和繁殖工作。
