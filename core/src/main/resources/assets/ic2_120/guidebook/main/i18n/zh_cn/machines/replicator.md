# 复制机

<BlockImage id="ic2_120:replicator" p:facing="north" scale="4" />

复制机使用存储的模式从 UU 物质复制物品。它需要 UU 物质流体和 EU 才能运行，并且必须能够访问存有所需模式的模式存储机。

## 能量与存储

- **能量等级：** 3
- **最大 EU 输入：** 512 EU/t
- **EU 存储：** 400,000 EU
- **UU 物质储罐：** 16,000 mB
- **基础速度：** 每 tick 推进 5 uB 模板成本
- **基础耗电：** 512 EU/t

## 需求

要复制物品，复制机需要：

1. **红石信号：** 没有红石就不会工作。
2. **UU 物质：** 可从除正面外的任意面通过管道输入，也可从容器输入槽中的 UU 物质桶/单元倒入。
3. **EU：** 3 级持续供电。
4. **模式：** 唯一相邻模式存储机中选中的模板。
5. **输出空间：** 输出槽必须为空或能与结果堆叠。

## 工作方式

从相邻模式存储机选择模板，设置单次或连续模式，然后给复制机红石信号。机器会按照模板的 UU 成本逐步消耗 UU 物质，并在每 tick 消耗 EU 推进进度。进度达到模板成本后，输出 1 个物品。

单次模式产出 1 个物品后等待；连续模式会在红石、EU、UU 物质、选中模板和输出空间都满足时持续复制。取消按钮会清空当前进度和流体消耗余数。

## 槽位与自动化

- **输出槽：** 复制出的物品
- **容器输入槽：** UU 物质桶、UU 物质单元或装有 UU 物质的流体单元
- **容器输出槽：** 倒空后的桶或空单元
- **电池槽：** 3 级电池放电
- **升级槽：** 4 个槽位

支持超频、变压、储能、弹出、拉取、流体弹出和流体拉取升级。拉取升级可拉取 UU 物质容器；流体拉取升级可把 UU 物质流体拉入储罐；弹出升级可推出输出物品。

## 完整链路

复制机是 UU 物质生产链的最后一个环节：**物质生成机**生产 UU 物质，**模式扫描机**创建模式，**模式存储机**存储模式，**复制机**最终产出物品。该系统允许您复制任何可扫描的物品，从基础资源到稀有材料。
