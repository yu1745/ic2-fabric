# 复制机

<BlockImage id="ic2_120:replicator" p:facing="north" scale="4" />

复制机使用存储的模式从 UU 物质复制物品。它需要 UU 物质流体和 EU 才能运行，并且必须能够访问存有所需模式的模式存储机。

## 能量与存储

- **EU 存储**：400,000 EU
- **等级**：3 级（HV，1024 EU/packet）

## 需求

要复制物品，复制机需要：

1. **UU 物质流体**：通过流体管道供应，或直接从相邻的物质生成机获取。
2. **EU**：为复制过程提供电力。
3. **模式**：来自相邻模式存储机的已存储模式。

## 工作方式

1. 确保复制机可以获取 UU 物质（通过流体管道或相邻的物质生成机）。
2. 在复制机旁连接一个存有所需模式的模式存储机。
3. 提供 EU 电力。
4. 从可用模式中选择要复制的物品。
5. 复制机消耗 UU 物质和 EU 来生产物品。

## 使用

复制机是 UU 物质生产链的最后一个环节：**物质生成机**生产 UU 物质，**模式扫描机**创建模式，**模式存储机**存储模式，**复制机**最终产出物品。该系统允许您复制任何可扫描的物品，从基础资源到稀有材料。
