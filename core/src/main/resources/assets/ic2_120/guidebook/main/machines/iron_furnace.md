# Iron Furnace

<BlockImage id="ic2_120:iron_furnace" p:facing="north" p:active="true" scale="4" />

The Iron Furnace is an upgraded version of the vanilla stone furnace that smelts items 1.5 times faster. It burns the same furnace fuels (coal, charcoal, lava buckets, etc.) but processes items more quickly, making it an early-game improvement before transitioning to electric machines.

## Slots

- Top slot: input item to smelt
- Bottom slot: fuel
