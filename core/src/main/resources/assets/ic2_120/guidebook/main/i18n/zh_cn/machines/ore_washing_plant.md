# 洗矿机

<BlockImage id="ic2_120:ore_washing_plant" p:facing="north" p:active="true" scale="4" />

洗矿机是矿石处理线中需要用水的中间步骤。它接收打粉机产出的粉碎矿石，消耗水和 EU，将其洗成纯净粉碎矿石，并产出洗矿副产品；之后可以直接烧炼，或继续送入热能离心机处理。

它也有沙砾配方，可以把沙砾洗成石粉。

## 运行

- **等级：** 1
- **EU 存储：** 16,000 EU
- **最大输入：** 高压升级前为 32 EU/t
- **能量消耗：** 推进进度时为 16 EU/t
- **处理时间：** 500 tick，即 25 秒
- **总耗能：** 升级前每次 8,000 EU
- **耗水量：** 每次 1,000 mB
- **水箱容量：** 8 桶，即 8,000 mB

只有输入物品存在洗矿配方、所有配方输出槽都能接收完整产物、机器有足够 EU，并且水箱足以供应下一段进度时，机器才会运行。如果输入为空、配方无效，或输出槽被堵满，进度会重置为 0。

## 配方

每个粉碎矿石配方消耗 1,000 mB 水，并产出 1 个纯净粉碎矿石、1 个石粉和一种副产品：

- 粉碎铜矿石 -> 纯净粉碎铜矿石、2 个小撮铜粉、石粉
- 粉碎锡矿石 -> 纯净粉碎锡矿石、2 个小撮锡粉、石粉
- 粉碎铁矿石 -> 纯净粉碎铁矿石、2 个小撮铁粉、石粉
- 粉碎金矿石 -> 纯净粉碎金矿石、2 个小撮金粉、石粉
- 粉碎铀矿石 -> 纯净粉碎铀矿石、2 个小撮铅粉、石粉
- 粉碎铅矿石 -> 纯净粉碎铅矿石、3 个小撮硫粉、石粉
- 粉碎银矿石 -> 纯净粉碎银矿石、2 个小撮银粉、石粉
- 沙砾 -> 石粉

完整矿石处理链通常是：矿石或粗矿 -> 打粉机 -> 粉碎矿石 -> 洗矿机 -> 纯净粉碎矿石。之后，纯净粉碎矿石可以直接烧炼成锭；如果存在对应热能离心机配方，也可以先离心得到额外粉类产物。

## 槽位

- **矿石输入：** 有洗矿配方的粉碎矿石或沙砾。
- **水输入：** 水桶或 IC2 水单元。空桶和空单元会返回到空容器输出槽。
- **输出：** 三个配方输出槽，用于纯净粉碎矿石、副产品粉和石粉。
- **空容器输出：** 来自水输入槽的空桶或空单元。
- **电池：** 1 个可充电 IC2 电池物品，用于向机器放电。
- **升级：** 4 个升级槽。

Shift 点击会把升级送入升级槽，电池送入电池槽，有效洗矿输入送入矿石输入槽，水桶或水单元送入水输入槽。

## 升级与自动化

洗矿机接受超频、变压、储能、弹出、拉取和流体管道升级。超频升级会提高速度和 EU/t 消耗。高压升级会从基础 1 级输入限制开始提高可接受电压等级。储能升级会增加内部 EU 缓冲。

外部物品传输可以插入升级、1 个电池、有效配方输入，以及装水容器；只能从三个配方输出槽和空容器输出槽取出物品。弹出升级会把这些输出槽推入附近库存，拉取升级会把有效洗矿输入拉入矿石输入槽。

内部流体水箱只接受水，不输出流体。流体管道或 Fabric 流体传输可以从暴露侧面向水箱注水；流体管道升级配置为接收模式时，也可以从相邻流体容器拉取水。
