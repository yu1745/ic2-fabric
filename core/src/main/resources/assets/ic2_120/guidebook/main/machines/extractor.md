# Extractor

<BlockImage id="ic2_120:extractor" p:facing="north" p:active="true" scale="4" />

The Extractor processes items by extracting valuable materials from them. Its primary use is extracting rubber from rubber wood and sticky resin, but it also handles various other extraction recipes.

## Output

- **EU Storage:** 800 EU
- **Tier:** 1
- **Energy Usage:** ~16 EU/t per operation

## Slots

- Left slot: input
- Right slot: output
