# 钢制转子

<ItemImage id="ic2_120:steel_rotor" scale="4" />

钢制转子是用于风力动能发生机和水力动能发生机的高级转子。

- **风力**：半径 4，倍率 3×
- **水力**：半径 2.0，倍率 3×

钢制转子提供了优异的性能，能够在中后期游戏中稳定输出大量动能。
