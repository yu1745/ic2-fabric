# Solid Canning Machine

<BlockImage id="ic2_120:solid_canner" p:facing="north" p:active="true" scale="4" />

The Solid Canning Machine fills or empties solid items into and from containers. It is used for tasks such as filling fuel cans, loading ammunition, and other solid-based canning operations.

## Output

- **EU Storage:** 416 EU
- **Tier:** 1

## Slots

- Top row (3 slots): container / fuel / filler slots
- Bottom row (3 slots): output slots
