# Fluid/Solid Canning Machine

<BlockImage id="ic2_120:canner" p:facing="north" p:active="true" scale="4" />

The Fluid/Solid Canning Machine combines the functionality of both the Solid Canning Machine and the Fluid Canner. It can fill or empty both solid items and fluid containers, making it a versatile machine for all canning needs.

## Output

- **EU Storage:** 832 EU
- **Tier:** 1

## Slots

- Top row (4 slots): container / ingredient slots
- Bottom row (4 slots): output slots
