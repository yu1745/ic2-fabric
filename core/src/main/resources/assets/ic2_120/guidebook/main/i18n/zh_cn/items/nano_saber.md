# 纳米剑

<ItemImage id="ic2_120:nano_saber" scale="4" />

<ItemLink id="ic2_120:nano_saber" /> 是一把 3 级电动剑。右键可切换激活状态；激活且有电时，每次命中造成 21 点高伤害。未激活时，或激活但 EU 耗尽时，则退化为原版剑，伤害仅为 5 点。

## 属性

纳米剑结合了 160,000 EU 的内部缓冲、零原版耐久的 `ToolMaterial`，以及 -2.4 的攻击速度修饰符。`NanoSaberActive` 这个 NBT 布尔值用于追踪剑当前是否点亮。伤害是动态计算的：未激活或无电时为 `5.0`，激活且缓冲中至少剩余 1 EU 时为 `21.0`。

它没有原版耐久条；耐久条上显示的仅是 EU 缓冲。EU 用光后，剑依然可以挥动——只是不再造成 3 级武器的伤害，回落到未激活时的伤害值。

| 属性 | 值 |
|----------|-------|
| 等级 | 3（能量水晶） |
| 最大容量 | 160,000 EU |
| 原版耐久 | 无（由 EU 驱动） |
| 挖掘速度 | 1.0 |
| 挖掘等级 | 0（无等级挖掘加成） |
| 附魔亲和性 | 15 |
| 攻击伤害（未激活或无电） | 5 |
| 攻击伤害（激活且有电） | 21 |
| 攻击速度修饰符 | -2.4 |
| 每次命中 EU 消耗（激活时） | 400 EU |

每次命中的消耗由代码常量 `ENERGY_PER_HIT = 400L` 设定。空闲时剑也会待机耗电：放在快捷栏中为 4 EU/t，放在物品栏中约为 0.25 EU/t。

## 使用方式

### 激活

- 将 <ItemLink id="ic2_120:nano_saber" /> 拿在主手，右键切换 `NanoSaberActive` 这个 NBT 标志。动作栏会显示一条简短消息确认新状态。
- 再次右键即可关闭。关闭时也会重置剑的待机耗电累加器，并停止激活状态的动画。
- 剑的物品模型使用 `NanoSaberActive` 的值驱动激活动画（激活时 `ticker` 每 tick 推进，每 5 tick 推进 10 帧中的下一帧）。

### 战斗

- **激活且有电**——每次命中造成 21 点伤害。激活状态下的攻击每次消耗 400 EU。
- **未激活，或激活但无电**——命中回落到 5 点伤害（即基础 `DAMAGE_INACTIVE_TOTAL`）。剑不会拒绝挥动；它只是不再是一把 3 级武器。

激活/未激活的伤害分支通过 `getAttributeModifiers` 计算：攻击伤害修饰符在"激活且有电"时解析为 `21.0 - 1.0 = 20.0`，否则为 `5.0 - 1.0 = 4.0`。攻击速度修饰符固定为 `-2.4`。

### 附魔

纳米剑构建在真正的 `SwordItem` 与真正的 `ToolMaterial`（`NanoSaberMaterial`）之上，因此附魔流程照常生效。锋利、亡灵杀手、节肢杀手、火焰附加、抢夺、耐久、经验修补、扫荡之刃和消失诅咒都可以正常生效。注意：激活模式的伤害由 `Weapon modifier` 属性修饰符设定，与锋利无关——但锋利等附魔的加成仍会叠加到最终伤害之上。

### 充电

- 纳米剑暴露一个 3 级 EU 接口。可从任意 3 级电源为其补充：例如 <ItemLink id="ic2_120:energy_crystal" /> 充电、<ItemLink id="ic2_120:mfe" />，或物品栏中相应等级的电池。
- 当剑放在快捷栏中时，在物品栏中右键 <ItemLink id="ic2_120:energy_crystal" /> 即可将 EU 从水晶泵入剑中。
