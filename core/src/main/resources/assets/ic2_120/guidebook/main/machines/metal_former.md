# Metal Former

<BlockImage id="ic2_120:metal_former" p:facing="north" p:active="true" scale="4" />

The Metal Former shapes metals into various forms using three different modes: Rolling (produces plates), Extruding (produces cables and rods), and Cutting (produces wire and casings). The mode can be changed in the machine GUI.

## Output

- **Tier:** 1
- **Energy Usage:** ~20 EU/t per operation

## Slots

- Left slot: input
- Right slot: output
