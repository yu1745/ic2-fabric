# 紫外线灯

<BlockImage id="ic2_120:uv_lamp" scale="4" />

紫外线灯在通入 EU 后会加速附近作物的生长。它发射的紫外线能刺激周围植物的生长，是自动化农场的重要设备。

## 输出

- **EU 存储：** 10,000 EU
- **效果：** 加速一定范围内的作物生长
