# 紫外线灯

<BlockImage id="ic2_120:uv_lamp" scale="4" />

紫外线灯用于加速附近的 IC2 作物架作物，但真正产生加速效果需要安装超频升级。没有超频升级时，它仍然会耗电并显示为工作状态，但不会给作物增加生长进度。

## 工作方式

- **储能：** 10,000 EU
- **升级槽：** 可放升级；其中超频升级决定灯的实际效果
- **有效超频上限：** 4 个
- **有效等级：** 超频数量 + 1
- **耗电：** 按有效等级对应的 EU/t 消耗
- **作物范围：** 每个方向 3 格

每个超频升级都会同时提高耗电等级和生长推动力度。作物 tick 到来时，正在工作的紫外线灯会给范围内尚未成熟的 IC2 作物增加额外生长点。超频越多，加速越明显，但也需要更高的 EU/t 和输入等级。

通常把它放在紧凑作物田的上方或侧面，再接入能承受当前超频等级的电缆。
