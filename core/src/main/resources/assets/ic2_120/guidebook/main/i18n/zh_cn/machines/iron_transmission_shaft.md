# 铁制传动轴

<BlockImage id="ic2_120:iron_transmission_shaft" scale="4" />

铁制传动轴用于沿直线传输动能（KU）。它是一种中级传动轴，可承受比木制更高、但低于钢制和碳纤维的功率。

## 吞吐量

铁制传动轴最多可承受 **128 KU** 的动能功率。

## 使用方式

- 将传动轴排成一条直线，用于从动能发电机向动能机器传输动能。
- 相邻放置时，动能会从一个传动轴传递到下一个。
- 铁制传动轴在六个面均可连接。
- 铁制传动轴比木制更耐用，适用于通用动能系统。
