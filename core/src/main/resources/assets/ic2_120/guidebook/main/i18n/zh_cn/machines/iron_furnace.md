# 铁炉

<BlockImage id="ic2_120:iron_furnace" p:facing="north" p:active="true" scale="4" />

铁炉是原版石炉的升级版本，冶炼速度是石炉的 1.5 倍。它燃烧与熔炉相同的燃料（煤炭、木炭、岩浆桶等），但加工效率更高，是在过渡到电动机器之前的早期改良设备。

## 槽位

- 上方槽位：待冶炼的输入物品
- 下方槽位：燃料
