# 铁炉

<BlockImage id="ic2_120:iron_furnace" p:facing="north" p:active="true" scale="4" />

铁炉是早期的燃料炉升级版。它使用原版熔炉烧炼配方，但每个物品只需要 **160 tick / 8 秒**，而原版熔炉需要 200 tick。

## 燃料行为

大多数熔炉燃料在铁炉中的燃烧时间为原版的 **1.25 倍**。由于每次烧炼只需 160 tick，煤炭和木炭可以烧炼 **10 个物品**，而原版熔炉只能烧炼 8 个。

岩浆桶是特例：固定燃烧 **2,000 tick**，可烧炼 **12.5 个物品**。它仍然是有效燃料，但在铁炉里远不如原版熔炉划算。

铁炉会在内部保存烧炼经验。取出产物或破坏方块时会结算这些经验。

## 槽位

- **输入槽：** 任意有原版烧炼配方的物品
- **燃料槽：** 有效熔炉燃料
- **输出槽：** 烧炼结果

物品自动化会把可烧炼物品送入输入槽，把有效燃料送入燃料槽。常规抽取只暴露输出槽。
