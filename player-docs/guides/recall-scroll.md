# 传送卷轴

## 用途

传送卷轴可以绑定传送机，让你在野外直接传送回绑定的传送机位置。

## 使用方法

1. **绑定**：手持传送卷轴，右键点击传送机即可绑定
2. **传送**：手持已绑定的传送卷轴，右键使用即可传送回绑定的传送机位置

## 注意事项

- **同维度**：只能在同一个维度内传送，跨维度无法使用
- **消耗能量**：传送会消耗背包中电池（充电电池、能量水晶等）的电量，距离越远、身上物品越多，耗电越高
- **绑定后外观**：绑定后传送卷轴的图标会从空白地图变成已填充地图
- **锚点被破坏**：如果绑定的传送机被拆除了，传送仍然会执行，但你会受到严厉惩罚
- 每人只能同时持有一张传送卷轴
