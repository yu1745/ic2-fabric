# IC2-120 玩家文档

## 橡胶树生成群系

橡胶树目前只会自然生成在以下群系中，找橡胶树优先去这些地方：

- `forest`：森林
- `flower_forest`：繁花森林
- `birch_forest`：白桦森林
- `dark_forest`：黑森林
- `taiga`：针叶林
- `old_growth_pine_taiga`：原始松木针叶林
- `old_growth_spruce_taiga`：原始云杉针叶林
- `jungle`：丛林
- `sparse_jungle`：稀疏丛林
- `bamboo_jungle`：竹林
- `swamp`：沼泽
