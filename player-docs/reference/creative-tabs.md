# 创造模式物品栏

主要分为三个栏位：

- `IC2 材料`：矿石、锭、导线、板材、粉末、橡胶等。
- `IC2 机器`：发电机、加工机、储电箱与充电台。
- `工具`：青铜工具、辅助工具、船、调试物品等。
- `种子`：三维为1的全部种子。

  **获取特定三维种子**：

  使用 `/ic2seed` 命令可直接获取指定三维的种子，需要生存模式开启作弊或创造模式。

  - `/ic2seed list` — 列出所有可用种子类型（含中文译名）
  - `/ic2seed <种子类型> <生长> <增益> <抗性>` — 获取指定三维的种子

  **不清楚种子英文名时**：
  1. 先用 `/ic2seed list` 查看完整列表，找到对应的中文名称
  2. 再使用 `/ic2seed <英文名> <生长> <增益> <抗性>` 获取种子
  3. 或直接使用 Tab 补全辅助输入

  **三维说明**：
  - **生长** (Growth)：作物生长速度，0-31
  - **增益** (Gain)：收获产量，0-31
  - **抗性** (Resistance)：环境适应能力，0-31

  **示例**：
  ```
  /ic2seed stick_maturity 31 31 31    # 完美成熟藤
  /ic2seed reed 20 15 10              # 自定义甘蔗
  ```
