# 核反应堆系统

核反应堆是 IC2 的高级发电方式，支持两种工作模式：
- **电模式**：燃料棒产热 → 堆温管理 → 散热片降温 → 发电输出 EU
- **热模式**：燃料棒产热翻倍 → 堆温管理 → 散热片加热冷却液 → 热能输出 HU

两种模式使用完全相同的组件摆放，热模式通过外层 5×5×5 结构自动判定。

---

## 多方块结构

核反应堆采用多方块结构：
- **中心**：核反应堆方块（`nuclear_reactor`）
- **扩展**：六面（上下前后左右）可各接触 0 或 1 个反应仓（`reactor_chamber`）
- **外壳**（热模式）：反应堆为中心的 5×5×5 立方体外壳

## 发电机制

### 核心公式

```
三角数(n) = (n² + n) × 2
每 1 output = 5 EU/t
最大输出：8192 EU/t（Tier 5）
```

### 铀燃料棒发电量

```
基础脉冲 = 1 + 联数 / 2
总脉冲 = 基础脉冲 × 联数 × 邻接加成
EU/t = 总脉冲 × 100
```

| 燃料棒类型 | 基准 EU/t（单格无邻接） |
|-----------|----------------------|
| 单联铀燃料棒 | 100 EU/t |
| 双联铀燃料棒 | 400 EU/t |
| 四联铀燃料棒 | 1,200 EU/t |

> 邻接中子反射器和其他燃料棒会大幅增加脉冲数。

### MOX 燃料棒

MOX 发电量随堆温变化：

| 堆温 | 效率 | 每脉冲 EU/t（单联） |
|------|------|-------------------|
| 0 | 0% | 100 |
| 2,500 | 25% | 200 |
| 5,000 | 50% | 300 |
| 7,500 | 75% | 400 |
| 10,000 | 100% | 500 |

## 堆温与安全

反应堆可贮存热量，最多 **10,000 HU**。

### 温度阈值

| 堆温 | 环境影响 |
|------|---------|
| > 4,000 | 5×5×5 范围内方块有几率着火 |
| > 5,000 | 5×5×5 范围内水有几率蒸发 |
| > 7,000 | 7×7×7 范围内生物受辐射伤害 |
| > 8,500 | 5×5×5 范围内方块有几率变成岩浆 |
| ≥ 10,000 | **核反应堆爆炸** |

> 防化服可免疫堆温 > 7,000 时的辐射伤害。

## 热模式（液冷反应堆）

热模式通过外层 5×5×5 外壳自动判定，需要至少 **2 个流体接口**。

### 电模式 vs 热模式

| 特性 | 电模式 | 热模式 |
|------|--------|--------|
| 发热量 | 正常 | **×2** |
| 发电量 | 正常 | **不发电** |
| 散热片蒸发 | 降低堆温 | **加热冷却液** |
| 流体需求 | 无 | 冷却液 + 热冷却液 |
| 散热片修复 | 自然修复 | **需冷却液** |

### 冷却液

- 罐容量：16,000 mB × 2（输入/输出）
- 转换比例：20,000 HU → 1,000 mB 热冷却液
- 散失热量定义：仅散热片的**自身蒸发**部分计入冷却液转换

### 热量守恒

```
燃料棒产热 × 2
→ 热量分配
→ 散热片蒸发（散失热量）
→ 加热冷却液
→ 未转换热量回灌散热片/堆温
```
