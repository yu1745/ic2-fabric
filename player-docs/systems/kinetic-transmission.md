# 动能系统（KU）

当前版本已实现独立于 EU 电网的 `KU` 网络。

## 核心参数

- 单位：`KU/t`
- 固定换算：`4 KU = 1 EU`
- 当前主要消费端：动能发电机

## 传动组件

| 组件 | 承载上限 | 损耗 |
|---|---:|---:|
| 木制传动轴 | 128 KU/t | 0 KU/格 |
| 铁制传动轴 | 512 KU/t | 2 KU/格 |
| 钢制传动轴 | 2048 KU/t | 1 KU/格 |
| 碳纤维传动轴 | 8192 KU/t | 0 KU/格 |
| 伞齿轮 | 2048 KU/t | 3 KU/格 |

## 连接与分配

- 传动轴只沿自身轴向连接。
- 伞齿轮用于 90° 换向。
- 多源多汇统一求解，按总供给减路径损耗分配。

## 风力动能发生机

### 基础

- 转子等级：木 / 铁 / 钢 / 碳
- 等级倍率：`1 / 2 / 3 / 4`
- 动能发电机每 tick 最大吞吐：`2048 KU/t = 512 EU/t`
- 标定点：碳转子在 `y=150`、晴天、`gustFactor=1.0` 时输出 `512 KU/t`（即 `128 EU/t`）

### 风力模型

- 主要受绝对 `Y` 高度影响。
- 高斯参数：`mu=150`、`sigma=35`、`floor=0.03`。
- 每区块每 `200 tick`（10 秒）更新一次阵风倍率。
- 阵风范围：`0.5 ~ 1.5`。
- 天气倍率：晴天 `1.0`，下雨 `1.2`，雷暴 `1.5`。

### 运行规则

- 前方阻挡或风机叶片重叠会卡住并停止输出。
- KU 从机器背面输出到动能网络。
- 转子只在“实际转动并产出 KU”时消耗寿命。
- 输出公式：`effectiveWind = mean(y) * gustFactor * weatherFactor`，`kuOut = baseKu * rotorMultiplier * effectiveWind`。
- 启动阈值按倍率同步放大（木/铁/钢/碳 = `1/2/3/4`），高阶转子需要更高风强。

### 转子寿命（晴天基准）

- 木：`3` 小时
- 铁：`24` 小时
- 钢：`48` 小时
- 碳：`168` 小时
