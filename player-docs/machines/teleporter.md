# 传送机 (Teleporter)

## 基础信息

- **电压等级**：无（不进 EU）
- **能量来源**：相邻的 MFE / MFSU 储电盒供电
- **升级槽**：4 个（支持超频、储能、变压）

## 功能

传送机可以将实体（玩家、生物）从一个坐标瞬间传送到另一个绑定的传送机。使用时需要相邻放置 MFE 或 MFSU 储电盒来提供能量。

## 使用步骤

1. **放置传送机+储电盒**：在传送机旁边紧贴放置 MFE 或 MFSU
2. **绑定**：手持遥控器分别右键两台传送机，将它们双向链接
3. **激活**：给传送机一个红石信号即可触发传送（范围内的实体将被传送）

你也可以使用**传送卷轴**来绑定某台传送机，在野外直接传送回来。

## 能量消耗

传送耗电由以下因素决定：

- **距离**：两点之间的距离越远，耗电越高
- **重量**：玩家身上的物品和装备越多，耗电越高
  - 基础重量 1000
  - 每件穿着的护甲 +100
  - 物品栏中的物品按堆叠比例增加重量
  - 重量上限 5100
- **目标类型**：
  - 玩家：按上述重量计算
  - 怪物：500 基础重量
  - 动物：100 基础重量

### 传送过程

1. 红石信号激活后，检测范围内最近的实体
2. 计算所需能量并开始充能（充能过程 1-2 秒，距离越远越长）
3. 能量充满后瞬间传送

## 注意事项

- **同维度**：只能在同一维度内传送
- **需要 MFE/MFSU**：必须相邻放置储电盒，直接从储电盒取电（绕过单 tick 输出上限）
- **冷却**：每次传送后有短暂冷却时间
- **传送范围**：传送机的检测范围可以通过升级提升
- **空间检查**：目标位置必须有足够空间，否则传送失败
- **双向绑定**：使用遥控器绑定两台传送机会建立双向链接
