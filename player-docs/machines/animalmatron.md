# 牲畜监管机 (Animal-Matron)

## 基础信息

- **电压等级**：Tier 1 (LV)
- **能量缓存**：10,000 EU
- **最大输入**：32 EU/t
- **最大输出**：0 EU/t（不输出）
- **流体槽**：水 2,000 mB / 抗草剂 2,000 mB
- **升级槽**：支持储能、超频、变压器升级

## 功能

牲畜监管机自动管理 9×9 范围内的动物，实现自动化养殖。

### 自动功能

| 功能 | 描述 |
|------|------|
| 喂食 | 每天自动投喂5个对应食物 |
| 成长控制 | 动物只能通过监管机喂食成长 |
| 繁殖控制 | 喂食10个食物后触发繁殖 |

### 流体需求

- **水**：用于维持动物正常生长（必需）
- **抗草剂 (Weed-EX)**：用于维持动物正常生长（必需）

> **注意**：缺少水或抗草剂时，监管范围内的动物将停止成长。

### 支持的动物

| 动物 | 食物 | 产物 |
|------|------|------|
| 猪 | 胡萝卜 | 猪肉 |
| 牛 / 哞菇 | 小麦 | 牛肉、皮革、牛奶 |
| 羊 | 小麦 | 羊肉、羊毛 |
| 鸡 | 小麦种子 | 鸡肉、羽毛、鸡蛋 |
| 兔子 | 胡萝卜、蒲公英 | 兔子肉、兔子皮 |
| 马 / 驴 / 骡 | 金苹果、金胡萝卜 | 皮革 |
| 羊驼 | 干草块 | 皮革 |

### 养殖机制

1. **每日喂食**：每只动物每天自动消耗 5 个对应食物
2. **成长条件**：累计喂食 10 个食物后，动物完成成长并可以繁殖
3. **繁殖限制**：监管范围内超过 32 只动物时停止繁殖
4. **繁殖重置**：繁殖一次后，计数器重置，需要再喂食 10 个食物才能再次繁殖

### 工作范围

- **范围**：9×9 区域（以机器为中心）
- **最大容量**：81 只动物（每格对应一只）
- **繁殖上限**：32 只动物

## 使用方法

1. **放置机器**：将牲畜监管机放置在动物养殖区域
2. **添加流体**：放入水桶和抗草剂桶（或对应流体容器）
3. **添加饲料**：在饲料槽中放入对应动物的食物
4. **添加能量**：放入电池或连接电网
5. **放置动物**：在监管范围内放入动物

## 与作物监管机对比

| 特性 | 作物监管机 | 牲畜监管机 |
|------|------------|------------|
| 主要功能 | 管理作物 | 管理动物 |
| 管理范围 | 9×9 区域 | 9×9 区域 |
| 流体需求 | 水 + 抗草剂 | 水 + 抗草剂（生长必需） |
| 消耗品 | 肥料 | 动物饲料 |

## 提示

- 确保水槽和抗草剂槽始终保持充足，否则动物将停止成长
- 饲料槽支持 7 种不同的食物，可同时养殖多种动物
- 使用超频升级可以提高工作效率
- 使用变压器升级可以连接更高电压的电网
