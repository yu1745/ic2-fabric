# 高炉 (Blast Furnace)

高炉是 IC2 中的炼钢设备。它从背面接收热量（HU），消耗铁质材料和压缩空气，产出钢锭（Steel Ingot）和炉渣（Slag）。

## 基础信息

- **电压等级**：Tier 1 (LV)
- **热量输入**：仅从**背面**接收热量（HU），需要热源机器紧贴背面摆放
- **温度范围**：0 ~ 1,700
- **工作温度**：> 1,400（达到 1,401 即开始工作）
- **压缩空气储罐**：8,000 mB（内部储罐，用压缩空气单元补充）
- **每钢锭耗气**：6,000 mB 压缩空气
- **升级槽**：2 个（仅支持弹出升级和抽取升级）

## 槽位

| 槽位 | 位置 | 说明 |
|------|------|------|
| 铁质材料（输入） | 左中 | 放入铁粉、粉碎铁矿石、纯净粉碎铁矿石、铁锭、铁矿石或深板岩铁矿石 |
| 压缩空气单元（输入） | 左下 | 放入压缩空气单元（Air Cell），机器自动抽取并返还空单元（Empty Cell） |
| 钢锭（输出） | 右上 | 生产出的钢锭 |
| 炉渣（输出） | 右中 | 副产品炉渣 |
| 空单元（输出） | 右下一格 | 消耗完的压缩空气单元变回空单元 |
| 升级槽 ×2 | 右上角 | 仅支持弹出升级（Ejector Upgrade）和抽取升级（Pulling Upgrade） |

### 铁质材料输入

高炉支持以下 6 种输入材料，每种材料消耗 1 个，均产出 1 个钢锭 + 1 个炉渣：

- 铁粉（Iron Dust）
- 粉碎铁矿石（Crushed Iron）
- 纯净粉碎铁矿石（Purified Iron）
- 铁锭（Iron Ingot）
- 铁矿石（Iron Ore）
- 深板岩铁矿石（Deepslate Iron Ore）

### 压缩空气输入

高炉需要压缩空气才能工作。把压缩空气单元（Air Cell）放进左下角的空气输入槽，机器会自动把空气抽进内部储罐，并返还空单元。

- 每个压缩空气单元提供 1,000 mB 压缩空气
- 内部储罐容量：8,000 mB（相当于 8 个压缩空气单元）
- 每加工 1 个钢锭消耗 6,000 mB 压缩空气

## 加热与温度机制

高炉本身不产热，需要在**背面**紧贴一个能输出热量（HU）的机器（如热传导发电机、电动加热机等）。

### 热量接收

高炉通过 `IHeatConsumer` 接口接收热量，只有**背面**（与机器正面相反的方向）才能接收 HU。机器正面是有颜色标记的那一面。

### 升温消耗

升温需要持续输入 HU，温度越高加热效率也越高（升温所需 HU 反而降低）：

| 温度区间 | 每 tick 消耗 HU |
|----------|----------------|
| 0 ~ 1,400 | 100 HU/t |
| 1,401 ~ 1,500 | 80 HU/t |
| 1,501 ~ 1,600 | 60 HU/t |
| 1,601 ~ 1,700 | 40 HU/t |

### 温度衰减

当 HU 输入不足时，高炉的温度会逐渐下降。下降速度同样随温度变化：

| 温度区间 | 每下降 1 所需 tick |
|----------|-------------------|
| 1,601 ~ 1,700 | 10 ticks（0.5 秒） |
| 1,501 ~ 1,600 | 16 ticks（0.8 秒） |
| 1,401 ~ 1,500 | 20 ticks（1.0 秒） |
| 0 ~ 1,400 | 24 ticks（1.2 秒） |

温度降到 0 后不再继续衰减。

## 加工速度

加工速度随温度提高而加快。温度越高，加工越快：

| 温度区间 | 加工时间 | 约需现实时间 |
|----------|---------|------------|
| 1,401 ~ 1,500 | 8,400 ticks | 7 分钟 |
| 1,501 ~ 1,600 | 6,000 ticks | 5 分钟 |
| 1,601 ~ 1,700 | 4,000 ticks | 3 分 20 秒 |

### 工作时温度冻结

当高炉正在加工时，温度冻结（不升不降），但需要持续消耗对应温度段的 HU 来维持。

如果 HU 输入不足以维持当前温度，加工进度中断，温度开始衰减，进度归零。

## 内部工作机制（给好奇的玩家）

高炉内部由以下几个组件协同工作：

- **热量接收组件**：从背面接收 HU，储存在内部缓冲中供升温使用
- **温度管理系统**：记录当前温度、升温进度和衰减进度，根据温度控制 HU 消耗量和加工速度
- **空气管理系统**：从输入槽抽取压缩空气单元，填入内部储罐（8,000 mB），返还空单元
- **配方系统**：匹配输入材料，消耗 6,000 mB 压缩空气和时间来完成炼钢
- **升级组件**：支持弹出升级（自动弹出输出槽产物）和抽取升级（自动从相邻容器抽取输入材料）

## GUI 界面说明

GUI 中各元素的含义：

- **顶部图标按钮**：显示支持的升级类型
- **温度条**：位于 GUI 中部偏右，显示当前炉温（鼠标悬停查看具体数值）
- **热量输入指示器**：当 HU 输入满足当前温度需求时亮起
- **温度达标指示器**：温度 > 1,400 时显示，表示可以开始工作
- **工作进度条**：显示当前加工进度
- **空气储罐显示**：左侧的柱状条显示压缩空气余量（鼠标悬停查看具体数值）
- **缺失空气提示**：空气不足时显示警示图标

## 没有的能力

- **不支持超频/储能/变压器升级**：仅支持弹出升级和抽取升级
- **不支持从外部给空气储罐输入流体**：压缩空气只能通过放入压缩空气单元的方式补充
- **不支持从外部输送物品**：没有自动输入功能，需配合管道 Mod 或抽取升级
- **不支持直接通入液体压缩空气**：不对外暴露空气储罐的流体接口（背面除外）

## 交互

- **右键**：打开 GUI，查看温度、空气余量和加工进度
- **扳手右键**：旋转机器，改变正面朝向（背面也随之改变）
- **扳手 Shift+右键**：反向旋转机器
- **扳手左键**：瞬间拆下机器，完好掉落（耗耐久）

## 建造建议

1. 高炉必须紧贴热源放置，热源贴在高炉**背面**。热源可以是斯特林发电机、热传导发电机等能输出 HU 的机器。
2. 建议搭配弹出升级，自动把钢锭和炉渣弹出到相邻的箱子。
3. 建议搭配抽取升级，自动从相邻箱子抽取铁质材料和压缩空气单元。
4. 如果使用流体管道 Mod 抽取空气，注意高炉的空气储罐仅通过侧面暴露，背面是热量接收面不会暴露流体接口。
