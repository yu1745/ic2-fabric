# 太阳能蒸馏机 (Solar Distiller)

## 基础信息

- **消耗**：无（免费运行）
- **产能**：每 2.5 秒（50 tick）蒸馏 1,000 mB
- **输入储罐**：10,000 mB（10 桶水）
- **输出储罐**：10,000 mB（10 桶蒸馏水）
- **电压等级**：无（不通电）
- **升级槽**：2 个（仅限流体弹出/流体抽入升级）

## 工作条件

太阳能蒸馏机**必须同时满足**以下所有条件才会工作：

| 条件 | 说明 |
|------|------|
| **主世界** | 只能在下界/末地之外的主世界工作 |
| **白天** | 游戏时间约 6:55 ~ 18:10（tick 333 ~ 11,750） |
| **晴天** | 下雨天不工作 |
| **天空视野** | 顶部不能有遮挡物（不透明方块会阻挡阳光） |
| **有水源** | 输入储罐需要有足够的水 |
| **未满** | 输出储罐未满（10,000 mB） |

## 工作方式

1. 从槽 0 读取水容器（水桶/水电池），将水存入输入储罐
2. 空容器自动放入槽 1
3. 满足工作条件时，机器自动将输入储罐的水蒸馏到输出储罐
4. 从槽 2 读取空电池，装满蒸馏水后放入槽 3

## 槽位

| 槽位 | 类型 | 说明 |
|------|------|------|
| 槽 0（水输入） | 输入 | 水桶/水电池/装水的流体单元 |
| 槽 1（空容器输出） | 输出 | 水消耗后留下的空桶/空单元 |
| 槽 2（空电池输入） | 输入 | 空单元用于装蒸馏水 |
| 槽 3（蒸馏水电池输出） | 输出 | 装好蒸馏水的电池 |
| 槽 4-5（升级） | 升级 | 2 个升级槽 |

## 生产效率

全天候理想条件下，每小时约产 14,400 mB（14.4 桶）蒸馏水。实际受天气和夜间影响。建议结合储罐使用，夜间不工作但白天积累的蒸馏水不会消失。

## 升级

仅支持两种流体升级：

- **流体弹出升级**：自动将蒸馏水输出到相邻管道/容器
- **流体抽入升级**：自动从相邻管道/容器抽取水

## 蒸馏水的用途

蒸馏水是核反应堆冷却液和[蒸汽发生器](steam-generator.md)的重要原料——蒸汽发生器使用蒸馏水可避免水垢积累，长期稳定运行。

## 交互

- **右键**：打开 GUI，查看水和蒸馏水存量
- **扳手右键**：旋转机器，改变朝向
- **扳手左键**：瞬间拆下机器，完好掉落
