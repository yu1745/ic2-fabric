# 动物屠宰机 (Animal Slaughterer)

## 基础信息

- **电压等级**：Tier 1 (LV)
- **能量缓存**：由升级决定（基础约 1,000 EU，每张储能升级 +10,000 EU）
- **最大输入**：32 EU/t（可经变压器升级提升）
- **升级槽**：4 个（支持储能、变压器、弹出升级）
- **剪刀槽**：1 个（消耗耐久）
- **电池槽**：1 个（可用电池供电）
- **输出槽**：15 个

## 功能

动物屠宰机自动扫描 9×9×3 范围内的成年被动动物，当数量达到 32 只时自动屠宰到剩余 16 只，收集掉落物到输出槽。

### 工作流程

1. **扫描**：每秒扫描一次 9×9×3 范围内的所有成年被动动物，消耗 1 EU/只
2. **判断**：动物数量 >= 32 只时启动屠宰，屠宰到剩余 16 只停止
3. **屠宰**：消耗剪刀 1 点耐久，消耗 20 EU/掉落物堆叠，击杀动物并收集掉落物
4. **输出**：掉落物进入 15 个输出槽，输出槽满则停止屠宰

### 支持的动物与掉落物

| 动物 | 掉落物 | 用途 |
|------|--------|------|
| 猪 | 2-3 猪排 | 食物 |
| 牛 | 2-3 牛肉 + 0-2 皮革（50% 概率） | 食物 + 材料 |
| 哞菇 | 2-3 牛肉 + 1-2 蘑菇（红/棕随机） | 食物 + 蘑菇 |
| 羊 | 1-2 羊肉 + 1 羊毛（未剪毛时掉落） | 食物 + 建材 |
| 鸡 | 1 鸡肉 + 0-2 羽毛 | 食物 + 箭矢材料 |
| 兔子 | 1 兔肉 + 0-1 兔子皮（50% 概率）+ 兔子脚（10% 概率） | 食物 + 材料 |
| 马 / 驴 / 骡 | 1-2 皮革 | 材料 |
| 羊驼 | 1-2 皮革 | 材料 |

### GUI 槽位说明

```
[输出槽 × 15]   [升级槽 × 4]   [电池]   [剪刀]
  (0-14)          (15-18)        (19)     (20)
```

- **输出槽（0-14）**：屠宰获得的掉落物自动存入，共 15 格
- **升级槽（15-18）**：安装储能/变压器/弹出升级
- **电池槽（19）**：放入充电电池为机器供电
- **剪刀槽（20）**：放入剪刀，屠宰时消耗耐久

## 搭建自动化养殖场

### 全套方案：牲畜监管机 + 动物屠宰机

将动物屠宰机与牲畜监管机配合使用，可以实现全自动养殖流水线：

```
                   牲畜监管机（繁殖+喂养）
                        │
                    9×9 养殖区
                   /          \
       [动物繁殖]              [动物屠宰机]
           │                        │
       饲料输入                  剪刀 + 电力
           │                        │
       自动喂食 ← ← ← ← ← ← 控制数量（32→16）
```

#### 步骤一：搭建养殖区域

1. 用围栏围出一块 9×9 的区域
2. 在区域中央放置 **牲畜监管机**（负责喂食和繁殖）
3. 在区域边缘放置 **动物屠宰机**（负责屠宰）

> 两台机器的工作范围均为 9×9（以各自机器为中心，屠宰机为 9×9×3 范围），确保养殖区在两台机器的重叠范围内。

#### 步骤二：配置牲畜监管机

1. 加入水和除草剂（虽然叫除草剂，但用在动物监管机里是杀虫剂效果，动物生长必需）
2. 加入对应动物的饲料（胡萝卜/小麦/小麦种子/金胡萝卜等）
3. 接入 EU 供电（电池或电网）

#### 步骤三：配置动物屠宰机

1. **安装升级**：建议至少安装 1 张储能升级，保证有足够的能量缓存
2. **放入剪刀**：屠宰机每次屠宰需要消耗剪刀耐久，务必保证剪刀槽有剪刀
3. **供电方式**：可通过电池槽放入充电电池，或从相邻导线取电
4. **接入弹出升级**（可选）：屠宰物自动弹出到相邻箱子，实现全自动收集

#### 步骤四：放入种畜

1. 放入 2 只同种动物（一公一母）
2. 牲畜监管机会自动喂食并触发繁殖
3. 当数量达到 32 只时，屠宰机自动启动，屠宰到剩余 16 只
4. 系统进入稳态：繁殖 16 只 → 屠宰 16 只 → 继续繁殖 → 循环

### 两种机器布局方式

屠宰机和监管机覆盖范围相同（9×9），但摆放方式决定了你能以多高的效率利用每台机器。以下两种方案各有优势。

#### 方案一：垂直叠放（共用一个 9×9 区域）

将屠宰机放在监管机正上方或正下方（不同 Y 高度），两台机器共用同一个 9×9 水平区域。监管机在上层喂食，屠宰机在下层屠宰。

```
监管机  →→ 覆盖 9×9（上层）
  ▲
  │  同一立柱，上下叠放
  ▼
屠宰机  →→ 覆盖 9×9（下层），扫描 Y 轴 ±3 格
```

- **优点**：空间紧凑，只占一个 9×9 的地皮
- **缺点**：一台屠宰机最多维持 32 只的产线，屠宰完就闲着

#### 方案二：对角线平铺（可无限水平扩展）

把监管机和屠宰机**按对角线放置**，让它们的覆盖范围错开但保持重叠。

```
┌───────────── 9 ─────────────┐
│                             │
│  监管机                      │
│  (A)                        │
│              ┌────── 9 ─────┼──┐
│              │              │   │
│              │    屠宰机    │   │
│              │    (B)       │   │
│              │              │   │
│              │              │   │
└──────────────┼──────────────┘   │
               │                  │
               └──────────────────┘
                ← 对角线间距 9 格 →
```

监管机放在 (0, 0)，屠宰机放在 (9, 9)，使两者的 9×9 覆盖范围在对角线方向叠加。这样一台屠宰机覆盖的区域内可以有 4 台监管机，反之亦然。

**为什么更高效**：监管机只管喂食繁殖，可以轻松喂满远超过 32 只动物；屠宰机是瓶颈。用对角线布局，4 台监管机向同一个 9×9 区域塞动物，一台屠宰机就能维持满负荷屠宰。

**无限扩展**：按此模式在二维平面上网格化排列，每个屠宰机服务 2-4 台监管机，每台监管机服务 1-2 台屠宰机：

```
 [监]   [监]   [监]
    [屠]     [屠]
 [监]   [监]   [监]
    [屠]     [屠]
 [监]   [监]   [监]
```

- **优点**：机器利用率高，屠宰机不停工，每台机器的性能都吃满
- **缺点**：占地面积大，需要较多的围栏和供电

### 供电方案

动物屠宰机每扫描 1 只动物消耗 1 EU，每产出 1 组掉落物消耗 20 EU。以一个 32 只动物的养殖区为例：

- 每次扫描消耗：32 × 1 = 32 EU
- 屠宰 16 只动物的掉落物消耗：约 16 × 20 = 320 EU（按每只 1 组掉落物估算）
- 单次工作总消耗：约 352 EU

**推荐供电方案**：

| 方案 | 设置 | 说明 |
|------|------|------|
| 基础 | BatBox + 导线 | 适合初期，手动充电池 |
| 进阶 | 发电机 → BatBox → 屠宰机 | 自动供电，挂机运行 |
| 高级 | 接入电网（LV/MV） | 配合变压器升级支持更高电压 |

### 剪刀管理

屠宰机每屠宰 1 只动物消耗剪刀 1 点耐久。一把满耐久的铁剪刀可屠宰 238 只动物。

**剪刀耗尽时**：机器会自动停止屠宰，输出槽中已有物品保留，放入新剪刀后恢复工作。

**自动化补充剪刀**：当前版本暂不支持自动补充剪刀，建议定期检查剪刀槽。

## 升级指南

4 个升级槽可自由搭配：

| 升级 | 数量 | 效果 |
|------|------|------|
| 储能升级 | 1-4 张 | 每张 +10,000 EU 容量，减少电池更换频率 |
| 变压器升级 | 1 张 | 支持更高电压输入（接入 MV 电网） |
| 弹出升级 | 1-4 张 | 自动将掉落物输出到相邻箱子 |

**推荐配置**：

- **挂机方案**：2 张储能升级 + 1 张弹出升级 + 1 张变压器升级
  - 储能升级保证不会因缺电停机
  - 弹出升级自动把肉和材料送到相邻箱子
  - 变压器升级支持接入 MV 电网
- **经济方案**：1 张储能升级 + 1 张弹出升级
  - 最低配置即可稳定运行
  - 配合 LV 电网或电池供电

## 内部工作机制（给好奇的玩家）

动物屠宰机内部由多个组件协作完成自动化屠宰：

- **扫描组件**：每秒遍历 9×9×3 范围，统计受监管成年动物数量（仅包含 AnimalFoodMapping 白名单中的动物类型）
- **屠宰组件**：达到 32 只阈值后启动，每次屠宰前检查剪刀耐久和 EU 存量，消耗机械剪刀击杀动物并生成掉落物
- **放电组件**：从电池槽的充电电池抽取 EU，补充内部缓存
- **邻域输电组件**：从相邻导线或储电箱自动抽电，实现电网供电
- **升级组件**：支持储能扩容、电压提升和自动弹出

## 注意事项

- **屠宰机只处理成年动物**，幼年动物不会被计入扫描和屠宰
- **输出槽满时停止屠宰**（共 15 格，足够容纳多次屠宰的产物）
- **不消耗剪刀实体**，只消耗耐久，耐久耗尽时剪刀留在槽内变为损坏状态自动清空
- **过压保护**：LV 机器接入 MV/HV 电网需要变压器升级，否则会损坏
- **无需手动干预**：放置好剪刀和供电后，屠宰机会自动维持动物数量在 16-32 只之间

## 常见问题

**Q：屠宰机为什么不工作？**

A：检查以下几点：① 养殖区是否有 32 只以上成年动物？② 剪刀槽是否有剪刀且耐久未耗尽？③ 是否有足够的 EU？④ 输出槽是否已满？

**Q：如何增加产量？**

A：建设更大的养殖区 + 多台牲畜监管机 + 多台屠宰机。每台屠宰机覆盖 9×9×3 范围，多个区域可并行运行。

**Q：动物不繁殖怎么办？**

A：确保牲畜监管机有充足的食物、水和除草剂。**缺少水或除草剂时，动物会停止成长**，不成长就不会繁殖。屠宰机只负责屠宰，不负责繁殖，繁殖需要牲畜监管机。
