# 发酵机 (Fermenter)

## 基础信息

- **机器类型**：耗热机器
- **电压等级**：Tier 1 (LV)
- **热量缓存**：40,000 HU
- **流体输入缓存**：10,000 mB（生物质）
- **流体输出缓存**：10,000 mB（生物燃料）
- **升级槽**：2 个（仅限流体弹出升级 / 流体抽入升级）

## 怎么用

发酵机消耗**生物质**和**热量**，生产**生物燃料**和**肥料**。它是一台热机，背后必须紧贴热源（如流体加热机、电力加热机或固体加热机）才能工作。

### 摆放

发酵机的**背面**（即出线口对应的那一面）必须紧贴一台加热机。加热机会向发酵机背面输送热量（HU），发酵机内部有 40,000 HU 的热量缓存。

### 投料

把盛装生物质的容器放进**左上角槽位**：

- 生物质桶
- 生物质单元（Biomass Cell）
- 装有生物质的流体单元（Fluid Cell）

机器会自动把里面的生物质抽到内部液体缓存池，空容器会进入右边相邻的**空容器槽**。

### 产出

- **生物燃料**：左下方的**满容器槽**输出装有生物燃料的桶/单元
- **肥料**：右下角的**肥料槽**产出肥料

### 容器处理流程

```
输入槽（生物质桶/细胞） → 机器抽取生物质 → 空容器槽
空容器槽 （空桶/空单元） → 机器注满生物燃料 → 满容器输出槽
```

你可以把空容器放在中间槽，让机器自动完成"注液"操作，省去手动倒液的步骤。

## 生产效率

每批次处理耗时 2 秒（40 tick），消耗与产出为：

每批次需要 4,000 HU，分摊到 40 tick 就是 **100 HU/t**。只有热源持续供应至少 100 HU/t 时，发酵机才能保持每 2 秒一批的全速运转。

如果热源输入不足（比如只有 20 HU/t），机器不会"降速工作"，而是**等热量攒够 4,000 HU 才启动一批**——相当于每 200 tick（10 秒）才处理一批。热量缓存上限 40,000 HU，最多可缓存 10 批的量。

| 项目 | 每次消耗 | 每次产出 |
|------|----------|----------|
| 热量（HU） | 4,000 HU | - |
| 生物质 | 20 mB | - |
| 生物燃料 | - | 400 mB |
| 肥料 | - | 每消耗 1,000 mB 生物质产出 1 个 |

计算一批生物质的全程投入产出：

| 投入 | 产出 |
|------|------|
| 1,000 mB 生物质（1 桶）+ 200,000 HU | 20,000 mB 生物燃料 + 1 个肥料 |
| 耗时：约 100 秒（50 批次） |

> 生物燃料是流体加热机的燃料，同时也可以在半流质发电机中发电。肥料可用于作物催熟。
>
> **场景一：外部热源满速**——发酵机由外部热源（如流体热交换机或电力加热机）提供 100 HU/t 全速运行，全部生物燃料用于发电。发酵机产 200 mB/s，每台半流质发电机烧生物燃料消耗 800/81 ≈ 9.877 mB/s。**200 ÷ 800/81 = 20.25 台**，半流质发电机每台产 16 EU/t，总计 **324 EU/t**。
>
> **场景二：流体加热机自循环**——用流体加热机提供 32 HU/t。发酵机攒够 4,000 HU 需 125 tick 一批，产 400 mB，即 3.2 mB/tick = 64 mB/s。扣除流体加热机自耗 10 mB/s，净余 54 mB/s。**54 ÷ 800/81 = 5.4675 台**，约 **87.48 EU/t**。适合不想拉外部电线的独立供能场景。

## 槽位布局

| 槽位 | 类型 | 说明 |
|------|------|------|
| 左上（槽 0） | 输入 | 生物质桶/生物质单元/装满生物质的流体单元 |
| 左中（槽 1） | 输入 | 空容器（机器注油后放入下方满容器槽） |
| 右中（槽 2） | 输出 | 空容器（从生物质容器中抽取后留下的空容器） |
| 左下（槽 3） | 输出 | 满容器（装好生物燃料的桶/单元） |
| 右下（槽 4） | 输出 | 肥料 |
| 右底（槽 5-6） | 升级 | 仅限流体弹出升级 / 流体抽入升级 |

## 升级支持

发酵机只支持两类升级：

- **流体弹出升级**：自动将内部生物燃料输出到相邻的流体容器/机器
- **流体抽入升级**：自动从相邻的流体容器/机器抽取生物质

不支持超频升级、储能升级、变压器升级等其他升级。

## 流体交互

发酵机两侧的流体储罐容量均为 10,000 mB：

- **输入储罐**：只接受生物质，不可抽取
- **输出储罐**：只储存生物燃料，可被外部抽取

安装流体弹出升级后，机器会自动把生物燃料推送到相邻的流体容器或管道。

安装流体抽入升级后，机器会自动从相邻的流体容器或管道抽取生物质。

## 内部工作机制（给好奇的玩家）

发酵机的工作周期由以下几个步骤构成：

1. **容器处理**：从槽 0 读入生物质容器，将生物质提取到输入储罐，空容器放入槽 2
2. **热量积累**：从背面加热机接收 HU，存储在 40,000 HU 的内部缓存中
3. **发酵反应**：每 2 秒触发一次，检查是否有足够的热量（4,000 HU）和生物质（20 mB），同时输出储罐有足够空间
4. **产物分配**：生成 400 mB 生物燃料到输出储罐，每累积消耗 1,000 mB 生物质产出 1 个肥料
5. **容器注油**：从槽 1 读入空容器，用输出储罐中的生物燃料注满，放入槽 3

四个条件**缺一不可**：热量不够、生物质不够、输出储罐满了、或者正在处理中，机器都不会继续下一批次。

## 交互

- **右键**：打开 GUI，查看生物质/生物燃料存量、热量缓存和肥料进度
- **扳手右键**：旋转机器，改变背面朝向
- **扳手 Shift+右键**：反向旋转机器
- **扳手左键**：瞬间拆下机器，完好掉落（耗耐久）

## 没有的能力

- **不支持 EU 输入**：不是电力机器，不通电
- **不支持超频**：装不了超频升级，加工速度固定为 2 秒一批
- **不是发电机**：不产生 EU 也不产生 HU，只消耗 HU 加工流体

## 搭建建议

1. 发酵机必须置于加热机前方（加热机背面贴发酵机背面）
2. 流体加热机使用生物燃料，发酵机产出生物燃料——可以用发酵机给流体加热机供燃料，形成闭环
3. 配合流体弹出升级，可以把生物燃料自动送入流体加热机或储罐
4. 配合流体抽入升级，可以把生物质从外部储罐自动抽入发酵机
