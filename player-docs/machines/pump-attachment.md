# 泵附件 (Pump Attachment)

## 基础信息

- **类型**：管道（自带抽取功能）
- **材质**：青铜 / 碳纤维
- **流量**：青铜 0.4 桶/秒 / 碳纤维 0.8 桶/秒（等效于本材质的微型管道）
- **尺寸**：微型（tiny）
- **方向**：有朝向，必须正面朝向目标方块

## 它是做什么的

泵附件首先是一节**微型管道**，然后才是抽取入口——它能像普通管道一样传输流体，同时正面朝向的方块会被它主动抽取。

## 怎么用

### 放置

泵附件有方向性，必须**正面朝向**你要抽流体的方块：

```
 储罐/机器    ←   泵附件
（有流体）   正面
```

- **正面**：只连接非管道的流体容器（储罐、机器），不连接其他管道
- **背面和侧面**：连接普通管道，构成传输路径

### 一格工作

泵附件自身就是**一段微型管道**，所以不需要额外拉管道也能用——正面贴源，背面贴目标，一个方块完成传输：

```
 源储罐  →  泵附件  →  目标储罐
（有流体）  正面朝左    
```

也可以在背面接普通管道延长，搭建长距离运输网络。

### 设置过滤（可选）

右键泵附件打开 GUI：

- 手持含流体的容器（桶、流体单元等）点击过滤槽 → 设置只抽取该种流体
- 空手点击过滤槽 → 清除过滤，抽取任意流体

不过滤时，泵附件抽取目标方块当前含有的任意流体。

### 常见用途

- **焦炭窑**：贴在炉篦正下方抽杂酚油，接去储罐或半流质发电机
- **抽水**：贴在储罐侧面接管到需要水的地方
- **机器排液**：贴在机器流体输出面接管到储罐

## 内部工作机制

泵附件本身是一个 tiny 尺寸的管道，额外带一个**定向抽取功能**：

- 每个 tick，管道网络扫描所有泵附件，把正面朝向的容器作为流体来源
- 从来源抽取流体，沿管道路径传输到接收端
- 一个网络内同时只能传输一种流体，混流会停机

## 没有的能力

- **不能作为接收端**：泵附件只抽不吸，接收流体用普通管道即可
- **不能过滤方向**：正面固定朝向抽取目标，不能改成其他面

## 交互

- **右键**：打开 GUI 设置过滤样本
- **扳手右键点击某面**：切换该面连接开关
- **扳手 Shift+右键**：切换透明模式观察内部流体
