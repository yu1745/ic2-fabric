# 拴绳动能发生机 (Leash Kinetic Generator)

## 基础信息

- **电压等级**：Tier 1
- **输出**：动能（KU）
- **动能上限**：512 KU/t（单次生成）
- **输出面**：所有六面均可输出 KU
- **升级槽**：无（不支持任何升级）
- **物品槽**：无（没有机器内部物品栏）

## 它是做什么的

拴绳动能发生机把动物绕机器的圆周移动转换成动能（KU），输出到传动网络。

简单说：你把一只动物拴在机器上，然后动物绕圈跑——机器就把跑圈的能量变成 KU。

## 怎么用

### 安装与连接

1. **放置机器**：像普通机器一样放地上
2. **准备拴绳**：手里拿着栓绳
3. **拴住动物**：先用栓绳右键拴住一只动物（让动物跟着你），然后拿着栓绳右键机器
4. **让它跑圈**：动物会被拴在机器上，你需要想办法让它绕着机器跑——比如用食物引诱它绕圈，或者找个合适的地方让动物自己走圈

### 解除拴绳

- **空手右键机器**：打开 GUI 查看状态
- **用栓绳右键已拴了动物的机器**：解开当前动物，把动物放开，还你一根栓绳
- **敲掉机器**：自动解开动物并收回栓绳

### 动能输出

机器产生的 KU 会通过所有六个面输出，可以直接紧贴[动能发电机](kinetic-generator.md)转换成 EU，也可以通过传动轴网络延长距离后再接入动能发电机。

```
拴绳动能发生机 → 传动轴 → 传动轴网络 → 动能发电机 → 电缆 → 储电箱
```

## KU 计算公式

机器每秒（20 tick）采样一次动物相对机器的角度，记录最近 40 次采样（约 2 秒）计算平均角速度。

```
KU/t = 8（基础值） + 2 × 平均角速度(°/s) + 水平拴绳距离(cm) ÷ 10
```

各项含义：

| 项目 | 说明 |
|------|------|
| **8 KU（基础值）** | 只要动物在运动就有的保底产出 |
| **2 × 角速度** | 动物绕圈转得越快产出越高，角速度是最近 40 tick 的平均值 |
| **拴绳距离 ÷ 10** | 拴绳的水平距离越长产出越高。每 10 格水平距离（注意是方块距离×100 换算成 cm）加 1 KU |

> 拴绳距离和角速度之间存在取舍：**半径越大，同样的移动速度下角速度越低**（周长更长，绕一圈更久）。玩家需要自己权衡——半径小则角速度快但距离加成低，半径大则距离加成高但角速度慢。最大值在中间某处取得。

**上限**：单次生成最高 **512 KU/t**。

**最低门槛**：平均角速度需要达到 **5°/s** 以上才会开始发电，原地不动或走得不够快都不行。

### 示例估算

假设动物以 30°/s 的平均角速度绕圈，拴绳水平长度为 3 格：

```
KU/t = 8 + 2 × 30 + 300cm ÷ 10
     = 8 + 60 + 30
     = 98 KU/t
```

## 拴绳系统

### 拴绳锚点

当你用栓绳右键机器拴动物时，机器会在自己位置生成一个**隐形的盔甲架**作为拴绳的锚点。动物就是被拴在这个隐形的盔甲架上，而不是直接系在机器上。

- 盔甲架不可见、无重力
- 这是内部机制，你不需要管它——只需要知道动物被拴在机器附近
- 解除拴绳或拆机器时，盔甲架会自动消失

### 拴绳范围

动物离机器的水平距离需要在 **10 格**以内，超出范围可能断绳。

要注意的是：拴绳距离是从动物位置到隐形盔甲架算的，盔甲架在机器方块中心偏下的位置（y+0.16），而动物的拴绳点大约在眼睛高度。这些细节你不需要记，只要知道动物别跑太远就行。

## 状态信息

机器没有操作型 GUI——所有运行数据通过 **Jade HUD** 查看：

| 信息项 | 说明 |
|--------|------|
| **状态** | 运行中 / 待机 |
| **动物** | 已拴绳 / 未连接 |
| **生成** | 当前每 tick 生成的 KU 量 |
| **输出** | 当前每 tick 输出的 KU 量 |
| **拴绳长度** | 动物到拴绳锚点的总距离（cm） |
| **角速度** | 动物绕圈的平均角速度（°/s） |

## 如何提高产出

想让机器发更多电，可以从三个方面入手：

1. **提高角速度**：让动物跑得更快、绕圈更顺畅。不同动物跑速不同——马比猪快，猪比鸡快。找一个跑得快的动物。
2. **增加拴绳距离**：把动物放远一些，拴绳水平距离越长产出越高。但要注意别超过 10 格范围。
3. **选择大个头动物**：拴绳长度算的是三维距离，个子高的动物（如马、牛）因为眼睛位置高，拴绳更长，间接略微增加产出。

### 实用技巧

- **马**：跑得快，角速度容易做高，是做动能的好选择
- **栅栏围圈**：用栅栏围一个圆形跑道，动物会沿着栅栏走，自然绕圈
- **避免卡住**：确保动物活动的空间没有障碍物，卡住就不会动，角速度降到 5°/s 以下就停止发电

## 关于升级

拴绳动能发生机**不支持任何升级**。不能装超频升级（加速没用，因为角速度取决于动物实际跑圈速度）、不能装储能升级（它没有内部缓存，产生的 KU 会立即等待输出）、不能装变压器升级（它输出 KU 不是 EU）。

## 内部工作机制（给好奇的玩家）

1. 每 tick 检查动物是否存活、拴绳锚点是否存在
2. 每 tick 计算动物相对于机器的角度
3. 记录过去 40 tick 的角速度，求平均值
4. 如果平均角速度 ≥ 5°/s，按公式计算 KU 并累加到待输出缓存
5. 其他机器（通过传动网络）可以从任意面提取 KU
6. 每 3 tick 同步数据到客户端用于渲染（GUI 和动物模型位置）

## 典型配置

```
拴绳动能发生机 → 传动轴 → 动能发电机 → 储电箱/直接用电
```

由于拴绳动能发生机一次最多产 512 KU/t，搭配动能发电机就是刚好：
- 512 KU/t ÷ 4 = 128 EU/t 输出
- 动能发电机最高输出 512 EU/t，完全够用

## 没有的能力

- **不支持升级**：不能装任何升级组件
- **无物品栏**：没有物品槽位，不处理任何物品
- **无 EU 输出**：只产生 KU，必须通过动能发电机转换成 EU
- **无流体**：不涉及任何液体
- **不能从外部充电**：它只生产 KU，不吃 KU

## 交互

- **右键（空手）**：打开 GUI 查看状态
- **右键（持栓绳）**：拴住动物 / 解除拴绳
- **扳手右键**：旋转机器，改变朝向
- **扳手 Shift+右键**：反向旋转
- **扳手左键**：瞬间拆下机器（耗耐久）
- **敲掉机器**：自动解除拴绳，动物恢复自由

## 建造建议

1. 初期可以在基地附近围一个圆形栅栏，把一只马拴上去
2. 一个拴绳动能发生机配一个动能发电机，就能获得稳定的 100+ EU/t 供电
3. 多只动物可以放多台机器并联，通过传动网络汇总到同一个动能发电机
4. 如果嫌动物麻烦，可以考虑其他动能来源：[手动动能发电机](manual-kinetic-generator.md)、[水力动能发电机](water-kinetic-generator.md)、[风力动能发电机](wind-kinetic-generator.md)
