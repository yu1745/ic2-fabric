# 蒸汽机 (Steam Generator)

## 基础信息

- **机器类型**：耗热机器
- **流体槽位**：水槽 10,000 mB / 蒸汽槽 100,000 mB
- **最大热输入**：1,200 HU/t
- **最高温度**：500°C（超过即爆炸！）
- **蒸汽膨胀比**：1 mB 水 → 100 mB 蒸汽
- **过热蒸汽阈值**：374°C（以上产出过热蒸汽）
- **水垢上限**：100,000 mB（仅普通水累积）
- **物品槽位**：无（纯流体机器）
- **升级支持**：不支持升级

## 概述

蒸汽机并不直接发电——它用**热量（HU）**把水烧成**高压蒸汽**，蒸汽再经由**蒸汽动能发生机**和**动能发电机**最终转化为 EU。它是从热能到电能这条产业链上的核心环节。

```
热源 → HU → 蒸汽机 → 蒸汽 → 蒸汽动能发生机 → KU → 动能发电机 → EU
```

四个方块组成一条完整的发电产线，缺一不可。

### 它可以做什么

- 消耗热量（HU）+ 水 → 产出大量蒸汽（流体）
- 温度 375°C 以上产出 **过热蒸汽**，能量密度翻倍
- 通过进水速率和压力调节，控制蒸汽产出效率

### 它不能做什么

- **不直接发电**：它产蒸汽，不是 EU。蒸汽要去蒸汽动能发生机才能变成 KU/KU 再进动能发电机才出 EU
- **不支持升级**：不能装超频、储能、变压器等升级
- **没有物品槽**：只走流体管道，不放东西
- **不收 EU**：它不是用电的机器，是吃热量的

## 搭建产线

### 第一步：热源

蒸汽机需要持续的热量输入。六面均可接收热量（HU），把热源贴在它任意一面即可。

常见热源：
- **固体加热机**（Electric Heat Generator）：耗电产热，最稳定
- **流体加热机**（Fluid Heat Generator）：烧流体燃料产热
- **地热发电机**（Geo Generator）：烧岩浆产热

### 第二步：供水

用流体管道把水（或蒸馏水）导入蒸汽机。水可以从任意面进入。

- **普通水**：到处都有，但会产生水垢
- **蒸馏水**：不会产生水垢，长期运行推荐

水源选择：
- 无限水源
- 泵 + 水脉
- 太阳能蒸馏机（产蒸馏水）

### 第三步：蒸汽输出

蒸汽会从任意面自动输出到相邻的流体管道或容器。用管道把蒸汽导入**蒸汽动能发生机（蒸汽动能发电机）**即可。

蒸汽动能发生机需要放入**涡轮转子**（Steam Turbine）才能工作。

### 第四步：动能转电能

蒸汽动能发生机输出 KU（动能），再接**动能发电机**（Kinetic Generator）把 KU 转成 EU。

> 一条完整的发电流水线示意图：
> ```
> 热源 → [蒸汽机] → 蒸汽管道 → [蒸汽动能发生机] → 动能轴 → [动能发电机] → 导线 → 你的机器
> ```

## GUI 界面

蒸汽机的 GUI 没有物品槽位，只有参数控制和状态显示。

### 进水速率（Input Rate）

- 范围：0 ~ 1,000 mB/t
- 决定每 tick 消耗多少水
- 默认值：100 mB/t
- 四个按钮档位：±1 / ±10 / ±100 / ±1,000

> 进水速率越高，蒸汽产出越多，但冷却也越剧烈。要根据热源的供热能力来调，不要盲目拉满。

### 压力（Pressure）

- 范围：0 ~ 300
- 默认值：100
- 调高压力 = 提高沸腾温度 = 产出更高品质的蒸汽
- 三个按钮档位：±1 / ±10 / ±100

> 压力不是越高越好。高压力需要更高的温度才能有效产蒸汽，温度不够时热量全用来升温了，反而没蒸汽出来。

### 状态显示

- **温度**：当前系统温度（°C）
- **压力**：当前蒸汽压力（0-300）
- **进水速率**：当前每 tick 进水设定
- **蒸汽产出**：当前每 tick 蒸汽产出量
- **水量**：水槽存量 / 容量（10,000 mB）
- **蒸汽量**：蒸汽槽存量 / 容量（100,000 mB）
- **水垢**：当前水垢累积百分比
- **运行状态**：是否正在产蒸汽

## 温度系统

### 初始温度

蒸汽机的初始温度取决于所在生物群系：

| 群系类型 | 基准温度 |
|---------|---------|
| 炎热群系（沙漠、热带等） | 45°C |
| 温带群系（默认） | 25°C |
| 寒冷群系（雪原、冰原等） | 0°C |

温度永远不会低于群系基准温度。

### 加热

- 每 1 HU 输入 = 温度上升 0.001°C
- 空烧（没水时）：所有热量直接用于升温，没有上限
- 有水时：如果热量不足以把水加热到有效温度，热量也用于升温

### 冷却

- **自然冷却**：每 tick 下降 0.01°C，不低于群系基准
- **产汽冷却**：每消耗 1 mB 水，降温约 `(100 + 压力÷220×100) × 0.001°C`，最高每 tick 降温 2°C

### 超温爆炸

温度超过 **500°C** 会爆！

```
爆炸半径：10 格
地形破坏：不破坏地形
```

不会把周围炸出大坑，但范围内玩家和生物会受到伤害。请务必监控温度，不要让机器超温。

## 蒸汽类型

### 普通蒸汽（Steam）

- 温度低于 375°C 时产出
- 蒸汽动能发生机中转换率：2 KU / mB
- 涡轮耐久损耗：2/次

### 过热蒸汽（Superheated Steam）

- 温度达到 **374°C 以上**时自动切换
- 蒸汽动能发生机中转换率：**4 KU / mB**（翻倍！）
- 涡轮耐久损耗：1/次（损耗反而更低）

> 过热蒸汽的性价比极高——同水量产出翻倍的能量，涡轮磨损反而减半。尽量维持高温运行。

## 低压模式

当压力设为 **0** 且温度低于 **100°C** 时，机器进入低压模式：

- 不产蒸汽，直接把水排到相邻容器
- 这相当于一个"冷水旁路"，正常运行时不会有这种情况

## 水垢系统（Calcification）

### 水垢怎么来的

使用**普通水**时，每处理 1 mB 水就会累积 1 mB 水垢。**蒸馏水完全不会产生水垢。**

### 水垢满了会怎样

水垢达到 **100,000 mB** 后，蒸汽机**彻底停摆**：
- 不再接收热量（`receiveHeat` 返回 0）
- 不再产蒸汽
- 需要清理水垢才能恢复

### 如何避免

- **使用蒸馏水**：这是最根本的解决方案
- 定期维护（如果有水垢清除手段的话）

## 蒸汽积压处理

蒸汽槽（100,000 mB）满时，多余的蒸汽必须排出，否则：

- **1/10 概率**：小爆炸（半径 1 格，无地形破坏）
- **9/10 概率**：回收水（每 100 mB 蒸汽变回 1 mB 水，回到水槽）

蒸汽能回收成水，这不算浪费——只是暂时停了机。但如果持续积压，爆炸概率会反复触发。

所以务必确保蒸汽管道畅通，蒸汽动能发生机持续工作。

## 内部工作机制（给好奇的玩家）

蒸汽机的核心逻辑分三步：

1. **热量积累**：每 tick 从六面接收 HU，缓存在内部缓冲区（最多 1,200 HU/t），跨 tick 保持
2. **蒸汽产出**：如果有水 + 压力 > 0 或温度 >= 100°C，消耗水 × 膨胀比（100 倍）产出蒸汽
3. **温度管理**：产汽会降温，热量输入会升温，无热源时自然冷却

三个关键参数相互作用：
- **进水速率**：控制水消耗速度，越高产汽越快，但降温也越快
- **压力**：提高有效沸腾温度，间接影响产出效率
- **热输入**：热源提供的 HU 决定了你能把多少水烧开

### 有效温度计算

```
有效温度 = 100°C + (压力 / 220) × 100 × 2.74
```

例如压力 100 时，有效温度约 224°C。系统温度必须超过这个值才能稳定产蒸汽。

### 产汽后冷却量

```
冷却量(°C) = 耗水量(mB) × (100 + 压力×100/220) × 0.001
上限：2°C / tick
```

## 常见问题

### Q: 蒸汽机多久能满负荷产蒸汽？

取决于热源功率。1,200 HU/t 满负荷运行时，配合 1,000 mB/t 进水速率，每 tick 产 100,000 mB（100 buckets）蒸汽。实际上蒸汽槽只有 100,000 mB，1 tick 就满了，所以蒸汽**必须持续排出**。

### Q: 我应该用什么压力？

建议从 **100** 起步。压力太低效率差，压力太高需要稳定高温。找到热源和进水速率的平衡点后，再逐步调高压力。

### Q: 普通水和蒸馏水差多少？

蒸馏水没有水垢问题，可以长期无人运行。普通水最多处理 100,000 mB 就会卡死（约 1,000 buckets 水）。如果坚持用普通水，大约处理 100 buckets 水后水垢就过半了。

### Q: 温度上不去怎么办？

检查：
1. 热源是否在工作（有没有 HU 输出）
2. 进水速率是不是太高（水太多把温度压下去了）
3. 压力是不是太高（有效温度太高，系统达不到）

尝试：
- 加大热源功率
- 降低进水速率
- 适当降低压力

### Q: 蒸汽动能发生机不工作？

检查：
1. 蒸汽管道是否连接了蒸汽机和蒸汽动能发生机
2. 蒸汽动能发生机里是否放了涡轮转子（Steam Turbine）
3. 蒸汽动能发生机的蒸馏水槽是否满了（满了会阻塞）
4. 蒸汽动能发生机后面是否接了动能发电机
