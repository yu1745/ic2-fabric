# UU 物质系统 (UU-Matter)

UU 物质系统是 IC2 的终极物质转换方案，由四台机器组成完整流水线：**将 EU 转化为 UU 物质流体，再通过扫描学习物品结构，最后用 UU 物质精确复制出任何已学习的物品**。

## 系统总览

```
  EU + 废料(可选)              物品输入
       │                          │
       ▼                          ▼
  ┌───────────┐            ┌───────────┐
  │物质生成机  │──UU物质──▶│ 模式扫描机 │
  │(产 UU 流体)│   (流体)   │ (学习结构) │
  └───────────┘            └─────┬─────┘
                                 │ 样板数据
                                 ▼
                           ┌───────────┐
                           │ 模式存储机 │  ← 模式存储水晶(导入/导出)
                           │ (存储样板) │
                           └─────┬─────┘
                                 │ 样板数据
                                 ▼
  ┌───────────┐            ┌───────────┐
  │物质生成机  │──UU物质──▶│  复制机   │──产出物品
  │(产 UU 流体)│   (流体)   │ (复制物品) │
  └───────────┘            └───────────┘
```

### 常用摆放顺序

模式扫描机 — 模式存储机 — 复制机 — 物质生成机，一字排开，模式存储机紧贴扫描机和复制机，UU 物质通过流体管道供给扫描机和复制机。

### 核心概念：uB（微桶）

UU 物质的用量以 **uB（micro-bucket，微桶）** 计算。1 桶 = 1,000,000 uB（一百万微桶），也就是 1 mB = 1,000 uB。

物品的复制成本以 uB 为单位，配置文件可查看/修改每种物品的价格。

## 物质生成机 (Matter Generator)

把 EU 转换成 UU 物质流体。投入废料（Scrap）可以大幅降低能耗。

### 基础参数

| 项目 | 数值 |
|------|------|
| **电压等级** | Tier 3 (HV) |
| **能量缓存** | 4,000,000 EU |
| **最大输入** | 不限（无上限） |
| **流体储罐** | 10,000 mB |
| **升级槽** | 4 个 |
| **需要红石信号** | 是 |

### EU 消耗

| 模式 | 每 mB 消耗 | 每 mB 需要废料 | 等效加速 |
|------|-----------|---------------|---------|
| **纯 EU 模式** | 1,000,000 EU | 0 | 基准 |
| **废料加速模式** | 166,667 EU | 34 个 | 约 6 倍 |

放入废料后，机器自动识别并切换模式，每产 1 mB 消耗 34 个废料，同时 EU 消耗降至原来的约 1/6。

### 槽位

| 槽位 | 用途 |
|------|------|
| **槽 0 — 废料** | 放入废料（Scrap），加速生产 |
| **槽 1 — 容器输入** | 放入空桶 / 空单元 / 流体单元，自动灌装 UU 物质 |
| **槽 2 — 容器输出** | 灌装后的容器（UU 物质桶 / UU 物质单元） |
| **槽 3 — 电池** | 放入电池为机器供能（注意：一个槽位只能放 1 个电池） |
| **槽 4-7 — 升级** | 见下文升级说明 |

### 支持的升级

| 升级 | 支持 |
|------|------|
| **储能升级** | 是（增加内部 EU 缓存） |
| **变压器升级** | 是（提高电压耐受等级） |
| **流体管道升级** | 是（流体输入/输出） |
| **弹出升级** | 是（自动弹出容器输出槽） |
| **抽取升级** | 是（自动抽取容器输入槽） |
| **超频升级** | **不支持** |

> 物质生成机不支持超频升级——它的产能由输入功率和废料决定，不能通过超频加速。

### 工作机制

1. 需要**红石信号**才会运行
2. 内部有 EU 缓存和 UU 物质储罐
3. 每 tick 消耗尽可能多的 EU，累积进度到 1,954 时产出 1 mB UU 物质
4. 有废料时自动切换加速模式，同时消耗废料
5. 满罐后自动停止（不浪费 EU）
6. 背面（非正面方向）可通过流体管道输出 UU 物质

### 能量连接

- 正面（有颜色面）：不可连接
- 其他五面：可接收 EU 输入，输入功率不限

## 模式扫描机 (UU Scanner)

扫描物品，学习其物质结构并将其保存到相邻的模式存储机。只有白名单中的物品才能被扫描。

### 基础参数

| 项目 | 数值 |
|------|------|
| **电压等级** | Tier 3 (HV) |
| **能量缓存** | 200,000 EU |
| **最大输入** | 512 EU/t |
| **单次扫描耗时** | 165 秒（3,300 tick） |
| **单次扫描耗能** | 844,800 EU（256 EU/t × 3,300 tick） |
| **升级槽** | 4 个 |

### 槽位

| 槽位 | 用途 |
|------|------|
| **槽 0 — 物品输入** | 放入待扫描的物品 |
| **槽 1 — 电池** | 放入电池为机器供能 |
| **槽 2-5 — 升级** | 见下文升级说明 |

### 支持的升级

| 升级 | 支持 |
|------|------|
| **超频升级** | 是（加快扫描速度） |
| **储能升级** | 是（增加内部 EU 缓存） |
| **变压器升级** | 是（提高电压耐受等级） |

> 扫描仪不需要流体管道升级或弹出/抽取升级。

### 工作流程

1. 必须**相邻放置一个模式存储机**（且仅一个，不能多不能少）
2. 放入的物品必须在 UU 复制**白名单**中（配置文件 `uuReplication.replicationWhitelist` 中定义）
3. 机器自动检测相邻的模式存储机：
   - 如果没有 → 状态：`NO_STORAGE`（无模式存储机）
   - 如果没有放入物品 → 状态：`NO_INPUT`（无输入）
   - 如果物品不在白名单中 → 状态：`NOT_WHITELISTED`（未在白名单）
   - 如果能量不足 → 状态：`NO_ENERGY`（能量不足）
4. 扫描过程中进度条逐渐增加，同时消耗 EU
5. 扫描完成后，**该物品的复制样板自动存入相邻的模式存储机**
6. 同一种物品只需扫描一次，之后再次放入同一物品时，机器立即显示完成（`COMPLETE`）

### 关于白名单

可复制的物品由配置文件决定，默认可复制石头、矿物、木材、食物等大量物品。价格从几十 uB（圆石）到上亿 uB（附魔金苹果）不等。

如果需要添加或移除可复制物品，可修改配置文件中的 `uuReplication.replicationWhitelist` 列表，或通过游戏内命令操作。

## 模式存储机 (Pattern Storage)

存放 UU 复制样板的中转设备。自身不耗能、不支持升级，只是用来存数据。

### 基础参数

| 项目 | 数值 |
|------|------|
| **电压等级** | 无需供电 |
| **能量缓存** | 无 |
| **升级槽** | 无 |
| **容量** | 可存储**任意数量**的样板 |

### 槽位

| 槽位 | 用途 |
|------|------|
| **槽 0 — 模式存储水晶** | 用于导入/导出样板数据（见下文） |

### 放置要求

- 必须与扫描仪或复制机**相邻放置**（紧挨着，共享一个面）
- 一个扫描仪或复制机**只能连接一个**模式存储机（不能同时连接多个）

### 样板管理

打开模式存储机的 GUI，可以看到所有已存储的样板列表：

- **已存储的样板**：显示每种可复制物品的名称和 UU 成本（uB）
- **选中高亮**：当前选中的样板会被高亮，复制机会使用这个样板的配方
- **点击选择**：点击任意样板即可切换选中状态

### 模式存储水晶 (Crystal Memory)

模式存储机支持使用**模式存储水晶**来导入/导出样板数据：

- **导出**：在 GUI 中选中一个样板，将模式存储水晶放入槽位，点击导出按钮 → 样板数据写入晶片
- **导入**：将存有样板数据的模式存储水晶放入槽位，点击导入按钮 → 样板存入存储器

这样可以实现样板在不同位置之间的搬运和备份。

## 复制机 (Replicator)

消耗 UU 物质流体和 EU，按照模式存储机的配方复制出物品。这是整条产线的最后一步。

### 基础参数

| 项目 | 数值 |
|------|------|
| **电压等级** | Tier 3 (HV) |
| **能量缓存** | 400,000 EU |
| **最大输入** | 512 EU/t |
| **运行时能耗** | 512 EU/t |
| **流体储罐** | 16,000 mB |
| **基础速度** | 5 UB/tick（100 UB/秒） |
| **升级槽** | 4 个 |
| **需要红石信号** | 是 |

### 两种模式

| 模式 | 行为 |
|------|------|
| **单次模式（默认）** | 复制 1 个物品后自动停止，需要再次给予红石脉冲才能继续 |
| **连续模式** | 持续复制，只要有材料和空间就一直生产 |

在 GUI 中点击模式切换按钮切换。

### 槽位

| 槽位 | 用途 |
|------|------|
| **槽 0 — 物品输出** | 复制完成的物品（**不可手动放入**，仅输出） |
| **槽 1 — 容器输入** | 放入 UU 物质桶 / UU 物质单元，自动倒入内部储罐 |
| **槽 2 — 容器输出** | 倒空后的容器（空桶 / 空单元） |
| **槽 3 — 电池** | 放入电池为机器供能 |
| **槽 4-7 — 升级** | 见下文升级说明 |

### 支持的升级

| 升级 | 支持 |
|------|------|
| **超频升级** | 是（加快复制速度） |
| **储能升级** | 是（增加内部 EU 缓存） |
| **变压器升级** | 是（提高电压耐受等级） |
| **流体管道升级** | 是（流体输入 UU 物质） |
| **弹出升级** | 是（自动弹出物品输出槽和容器输出槽） |
| **抽取升级** | 是（自动抽取容器输入槽） |

### 工作流程

1. 需要**红石信号**才会运行
2. 必须相邻放置一个**模式存储机**（且其中必须有选中的样板）
3. 样板的复制成本以 uB 为单位，进度在 0 ~ 成本值之间增长
4. 每 tick 消耗 512 EU + 对应数量的 UU 物质流体
5. 进度达到成本值时，产出一个物品到输出槽
6. **微桶级精度**：流体消耗精确到微滴（droplet），不浪费
7. 输出槽满时自动停止（`NO_OUTPUT` 状态）

### 常见状态说明

| 状态 | 含义 |
|------|------|
| `NO_REDSTONE` | 未收到红石信号 |
| `NO_STORAGE` | 未找到相邻的模式存储机 |
| `NO_TEMPLATE` | 模式存储机中未选中任何样板 |
| `NO_FLUID` | UU 物质不足 |
| `NO_OUTPUT` | 输出槽满了或被占 |
| `NO_ENERGY` | 能量不足 |
| `RUNNING` | 正常运行中 |
| `COMPLETE` | 单次模式已完成复制 |

### 能量连接

- 正面（有颜色面）：不可连接
- 其他五面：可接收 EU 输入，最高 512 EU/t（可通过变压器升级提升）

## 完整搭建指南

### 基础布局示例

最简单的搭建方式：

1. **物质生成机** 和 **复制机** 紧邻放置，给物质生成机装**流体弹出升级**（或给复制机装**流体抽入升级**），UU 物质直接相邻传输，不需要拉管道
2. 如果距离远，也可以用**流体管道 + 泵附件**连接
3. **模式存储机** 同时紧邻 **模式扫描机** 和 **复制机**（三者呈 L 形或直线排列）
4. 模式扫描机和复制机各接一条 HV 导线供电

### 搭建步骤

1. **第一步：造物质生成机**，供 EU + 废料，开始产 UU 物质
2. **第二步：造模式存储机**，放在未来扫描仪和复制机都能相邻的位置
3. **第三步：造 模式扫描机**，紧挨着模式存储机，放入待扫描的物品
4. **第四步：扫描完成后，造复制机**，紧挨着模式存储机的另一侧
5. **第五步：给复制机供 UU 物质（流体管道）+ EU，开始复制**

### 进阶技巧

- **废料量产**：用回收机（Recycler）处理 cobblestone 等廉价物品批量生产废料，供给物质生成机
- **批量复制**：复制机切换到连续模式，配合弹出升级，可无人值守批量生产
- **样板搬运**：用模式存储水晶将样板从一条产线复制到另一条产线，避免重复扫描
- **超频加速**：扫描仪和复制机都支持超频升级，放入后可显著缩短扫描/复制时间
- **流体管道供料**：给复制机装流体管道升级，可以从相邻的储罐或管道网络自动吸入 UU 物质

## 组件一览

| 机器 | 功能 | 电压 | 耗能/产能 |
|------|------|------|----------|
| **物质生成机** | EU → UU 物质流体 | HV (Tier 3) | 1,000,000 EU/mB（有废料 166,667 EU/mB） |
| **模式扫描机** | 学习物品复制配方 | HV (Tier 3) | 256 EU/t，165 秒/次 |
| **模式存储机** | 存储复制样板 | 无 | 不耗能 |
| **复制机** | UU 物质 + EU → 物品 | HV (Tier 3) | 512 EU/t，速度取决于 UU 成本 |
