# 插件工作台设计计划书

> 注意：本文档为计划书，待审阅通过后实施。

## 1. 概述

为插件系统设计一个专用机器方块 — **插件工作台（Plugin Workbench）**，所有插件的安装与拆卸操作均在此机器上完成，不在手上直接操作。

## 2. 为什么需要专用机器

- **原版附魔的替代仪式感**：附魔需要附魔台，插件也应有对应的工业机器
- **消耗控制**：机器可以消耗 EU + 额外材料，比手上右键更有深度
- **拆卸风险**：机器可以控制拆卸成功率，增加策略性
- **升级路线**：机器本身可升级（基础/高级/量子），解锁更多功能

## 3. 机器设计

### 3.1 基本信息

| 属性 | 值 |
|---|---|
| 名称 | 插件工作台（Plugin Workbench） |
| 等级 | 2 |
| EU 容量 | 100k EU |
| EU/t 消耗 | 安装 128 EU/t，拆卸 256 EU/t |
| 尺寸 | 1×1×1（标准机器） |
| 配方 | 基础机械外壳 + 高级电路 + 拉普顿晶体 + 荧石粉×4 |

### 3.2 GUI 布局

武器居中放置，插件槽围绕武器呈五边形排列。背景为科技感能量回路装饰，从中心向外辐射。

```
+--------------------------------------------------+
|  插件工作台                    ⚡ 45,000 / 100,000 |
|                                                      |
|                 ╭─────────────╮                      |
|             ┌───┤   [③ 插件]  ├───┐                 |
|             │   ╰─────────────╯   │                 |
|        ╭────┤                      ├────╮           |
|        │    │     ╭───────────╮   │    │           |
|        │[④] │     │  [武器]   │   │[②]│           |
|        │    │     │  ═══════  │   │    │           |
|        ╰────┤     ╰───────────╯   ├────╯           |
|             │   ╭─────────────╮   │                 |
|             └───┤   [⑤ 插件]  ├───┘                 |
|                 ╰─────────────╯                      |
|            ╭─────────────╮  ╭──────────╮            |
|            │  [材料槽]   │  │ [输出槽] │            |
|            ╰─────────────╯  ╰──────────╯            |
|                                                      |
|     [⚡ 安装 5,120 EU]  [⚡ 拆卸 2,560 EU]           |
|                                                      |
|  已安装插件:                                         |
|  ✅ 威力插件 III  ｜ ✅ 火焰插件 II                   |
|  ○ 空插槽 × 2                                      |
+--------------------------------------------------+
```

**布局说明：**

- **中心**：武器槽，尺寸略大于普通槽位（32×32 → 40×40），带发光边框，视觉上突出为焦点
- **外围**：5 个插件槽呈等边五边形排列，每个间距相等、角度均匀（360°/5 = 72°）
- **底部**：左侧材料槽（消耗品），右侧输出槽（拆卸产物），尺寸为标准 32×32
- **背景纹理**：深色科技风底板 + 能量回路图案，中心到每个插件槽之间有发光的能量连接线，机器运行时连线有从中心向外流动的动画
- **右上角**：EU 存储条，实时显示
- **底部信息区**：已安装插件清单（带 ✅/○ 状态标记）+ 空余插槽数
- **按钮**：安装 / 拆卸，按钮上直接显示 EU 消耗量，按钮在操作进行中 disabled

### 3.3 容器设计

```
SimpleInventory(8):
  slot 0: 武器槽（中心）
  slot 1~5: 插件槽（五边形外围，按顺时针排列）
  slot 6: 材料槽（左下）
  slot 7: 输出槽（右下）
```

- 武器槽（slot 0）：特殊处理，不可被 Shift 移入插件/材料
- 插件槽（slot 1~5）：SlotSpec 限定只接受 `PluginItem`，空插槽在 GUI 上显示灰色虚线框
- 材料槽（slot 6）：SlotSpec 限定只接受指定材料（如绑定剂、冷却液等）
- 输出槽（slot 7）：SlotSpec 限定只可取出不可放入，点击「取出」按钮或 Shift 快速取出
- Shift 规则：通过 SlotSpec + SlotMoveHelper 配置，武器 → 武器槽，插件 → 最近的空插件槽

### 3.4 操作流程

#### 安装流程
1. 将武器放入中心武器槽
2. 将插件物品放入任意空插件槽（五边形上）
3. （可选）放入消耗材料
4. 点击「安装」按钮
5. 机器消耗 EU + 材料，插件物品被消耗，武器 NBT 写入插件数据
6. 对应的插件槽清空，武器信息区更新

#### 拆卸流程
1. 将已安装插件的武器放入中心武器槽
2. 从底部信息区点击要拆卸的插件（选中高亮）
3. （可选）放入消耗材料
4. 点击「拆卸」按钮
5. 机器消耗 EU + 材料，进行成功率判定
6. 成功：插件以物品形式进入输出槽，武器该插槽位释放
7. 失败：武器/插件均不损失，但 EU 和材料消耗不退

## 4. 安装与拆卸的消耗设计

### 4.1 安装消耗

| 方案 | 消耗材料 | EU 消耗 | 特点 |
|---|---|---|---|
| **A（推荐）** | 仅消耗 EU，不消耗额外材料 | 256 EU/t × 20t = 5,120 EU | 简单，鼓励频繁换装 |
| B | 消耗 EU + 通用材料（红石粉） | 1,280 EU + 1 红石粉 | 轻微消耗，防止无成本 |
| C | 消耗 EU + 对应插件等级的绑定材料 | 2,560 EU + 绑定剂×1 | 高等级插件需要更高成本 |

### 4.2 拆卸消耗与成功率

| 方案 | 基础成功率 | 消耗材料提升 | EU 消耗 |
|---|---|---|---|
| **A（推荐）** | 100% | 不需要 | 2,560 EU |
| B | 80% | 添加冷却液 → 95% | 5,120 EU |
| C | 50% | 添加冷却液+钻石 → 100% | 10,240 EU |

**推荐采用方案 A**（安装消耗 EU，拆卸 100% 成功率消耗 EU），原因：
- 玩家体验流畅，不 punitive
- 核心消耗在于插件本身的制作成本
- 拆卸再收费会让人不敢尝试新组合

如果要增加挑战性，可以用方案 B：基础 80% 成功率，通过材料提升到 100%。

## 5. 机器工作速度

| 操作 | 处理时间 |
|---|---|
| 安装 | 2 秒（40 tick） |
| 拆卸 | 3 秒（60 tick） |
| 动画 | EU 条闪烁，粒子效果 |

处理期间 GUI 锁定，机器持续消耗 EU，若中途断电则操作取消、材料不退。

## 6. 升级路线

| 等级 | 机器名称 | 解锁功能 | EU 容量 | 合成 |
|---|---|---|---|---|
| 基础 | 插件工作台 | 安装、拆卸 | 100k | 基础机械外壳 + 电路 |
| 高级 | 高级插件工作台 | + 批量操作 + 预览属性 | 1M | 基础版 + 高级合金 + 强化电路 |
| 量子 | 量子插件工作台 | + 无损跨武器转移插件 + 100% 拆卸 | 10M | 高级版 + 量子核心 + 铱板 |

## 7. Sync & 网络同步

- 使用 `TickLimitedSidedEnergyContainer`（复用现有模式）
- `SyncedData` 同步：武器槽物品、插件槽物品、材料槽物品、输出槽物品、进度、EU
- 使用 `PropertyDelegate` 同步操作进度和 EU（与太阳能面板模式一致）

## 8. Screen / ScreenHandler

- ScreenHandler：8 个槽位（1 武器 + 5 插件 + 1 材料 + 1 输出），使用 `SlotSpec` 定义规则
- Screen：Compose UI 绘制，背景使用纹理贴图显示能量回路图案
- 中心武器槽位使用 Compose 绘制突出效果（放大、发光边框）
- 五边形插件槽位通过 Compose 计算坐标（极坐标 → 屏幕坐标）精确定位
